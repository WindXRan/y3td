# 局外成长系统

> 从《项目主 GDD（初版）》拆分而来，便于按专题维护。

## 7. 局外成长系统

### 7.1 系统定位
- 局外长期成长固定由 `职业`、`装备`、`遗物` 三条主线组成，并统一通过局外存档长期累积。
- 三条线都要对局内产生影响，但职责必须明确：
  - `职业` 决定玩法方向与流派骨架。
  - `装备` 决定刷图构筑深度、词条质量与套装方向。
  - `遗物` 决定长期数值上限与高稀有追求目标。
- 局外成长提升的是 `选择宽度`、`开局质量` 和 `长期追求`，不能直接取代局内的 `F 羁绊`、`宝物`、`烙印` 与 `成长武器` 决策。
- 当前 runtime 已有 `章节 / 模式解锁 + 最近结果回流 + 存档骨架`，本章节定义的是后续要接入该骨架的正式局外设计。

### 7.2 三大成长线

#### 7.2.1 职业系统
- `职业` 是局外第一核心系统，负责定义玩家“这一把想怎么玩”。
- 职业本身 `不使用品质分级`，统一采用：
  - `职业解锁`
  - `职业等级`
  - `职业天赋树`
- 每个职业都应具备清晰的玩法标签，例如：
  - `普攻 / 连击`
  - `技能爆发`
  - `召唤 / 持续伤害`
  - `暴击 / 穿透`
  - `生存反击`
- 职业天赋树建议分为三层：
  - `基础层`：基础属性、资源效率、职业通用被动。
  - `分支层`：把同一职业导向不同 build，例如 `速攻流 / 暴击流 / 技能流`。
  - `核心层`：少量关键节点，用于强化职业标签，但不能直接代替装备和遗物的职责。
- 虽然职业 `没有品质`，但设计上仍应采用 `职业模板档位` 来区分定位、成本和上限。
- 推荐固定分为 3 类模板，并仅作为设计层和 UI 标签使用，不进入白绿蓝紫橙金体系：
  - `入门模板`
  - `主力模板`
  - `高上限模板`
- 这 3 类模板的职责分别为：
  - `入门模板`：机制直观、培养便宜、前期成型快，主要负责新玩家上手。
  - `主力模板`：机制中庸但强，成本略高于入门模板，目标是成为大多数玩家尤其是平民玩家的主力职业池。
  - `高上限模板`：机制特殊、构筑空间大、可整活，解锁金币更多、养成材料更多，主要服务熟练玩家和高投入玩家。
- `高上限模板` 的强度优势必须建立在 `高投入 + 正确构筑` 之上，不能做到“解锁即全面优于主力模板”。
- `主力模板` 应是综合性价比最高的一档，保证平民玩家可以长期投入且有爽感。
- `入门模板` 不能被做成废职业，必须保留稳定通关与教学价值。
- `金币` 可用于解锁普通职业，但价格应明显偏高，形成长期目标。
- `钻石` 可作为万能替代货币，用于补足职业解锁或天赋培养消耗，但不应成为性价比最高的常规投入方式。

#### 7.2.2 装备系统
- `装备` 是局外第二核心系统，主要通过 `刷关掉落` 获得，是长期毕业的主要来源。
- 局外装备与当前局内的 `成长武器` 不是同一系统：
  - `成长武器` 负责单局中的金币成长主轴。
  - `局外装备` 负责入局前的长期养成、词条构筑和套装方向。
- 装备建议采用固定多部位结构，首版可按 `5 部位` 规划，参考“圣遗物式”换装体验。
- 每件装备都包含：
  - `品质`
  - `主词条`
  - `副词条`
  - `套装归属`
- 装备词条采用随机模型：
  - `主词条类型` 随机。
  - `副词条类型` 随机。
  - `词条数值档位` 也随机。
- 装备支持 `洗练 / 重铸`：
  - 基础洗练用于重置随机词条结果。
  - 高级洗练可预留为后续版本，用稀有资源支持 `锁词条` 或 `定向保留`。
- 套装效果可以同时包含：
  - `纯数值强化`
  - `小幅机制强化`
- 机制型套装效果可以提升某类玩法手感，例如延长持续时间、提高触发稳定性、补一小段机制链条，但强度不应超过高稀有遗物。
- 装备品质应随着玩家刷的关卡和难度逐步抬升，形成明确的 `刷更高关 -> 掉更高品质装备` 反馈。
- 卡池中允许存在少量 `通用型强力装备`，服务于加速养成与补充泛用战力。
- 但同品质下，通用装备在 `特定流派适配度` 上固定弱于刷图专精装备约 `20%`：
  - 例如通用装备提供 `+100 力量 / +100 敏捷 / +100 智力`
  - 同级敏捷专精装备可提供 `+140 敏捷 / +60 力量 / +60 智力`
- 套装同样遵循这个原则：`通用套装更泛用，但不如刷图定向套装更极致。`

#### 7.2.3 遗物系统
- `遗物` 是局外第三核心系统，主要通过 `卡池` 获取，承担长期数值积累与高稀有追求目标。
- 遗物建议采用 `少量固定槽位`，避免数量过多后压过职业与装备的存在感。
- 遗物分为两类职责：
  - `低稀有遗物`：提供平滑、稳定、差距不夸张的数值加成。
  - `高稀有遗物`：在数值之外，提供有限但明显的机制强化。
- 遗物重复获取后，应能转化为专属资源，用于：
  - `升星 / 升阶`
  - `保底进度`
  - `商店兑换`

### 7.3 品质分级制度
- 全局常驻养成内容统一采用以下 6 档品质：
  - `普通（白）`
  - `优良（绿）`
  - `精良（蓝）`
  - `史诗（紫）`
  - `传奇（橙）`
  - `传说（金）`
- `职业` 不参与品质分级。
- `装备` 与 `遗物` 统一使用该品质体系，但两者的获取和成长职责不同。

#### 7.3.1 遗物品质规则
- `紫色及以下遗物`：只提供纯数值加成，不承担玩法机制变化。
- `橙色遗物`：获取难度始终较高，在较强数值外，可附带 `轻度机制强化`。
- `金色遗物`：获取难度最高，可提供 `更强但仍可控的机制强化`。
- 橙金遗物的机制强化应遵循：
  - 强化已有职业或构筑方向。
  - 不能单件遗物直接改写一个职业的玩法定位。
  - 不能让职业和装备系统失去存在意义。

#### 7.3.2 装备品质规则
- 装备品质与关卡进度强绑定，玩家刷的章节、难度越高，高品质装备的产出占比越高。
- 装备的品质差异主要体现在：
  - `基础词条区间`
  - `可出现的副词条质量`
  - `洗练上限`
  - `套装效果强度`
- 装备的机制强化允许存在，但总体应低于橙金遗物的机制权重。

### 7.4 局外货币

#### 7.4.1 金币
- `金币` 是局外常规养成货币，负责大部分“可肝可攒”的稳定成长。
- 金币的主要用途包括：
  - 解锁普通职业
  - 提升职业等级
  - 点亮职业天赋树中的普通节点
  - 装备基础强化
  - 装备基础洗练
  - 购买普通养成资源
- 金币的设计目标是：即使不依赖高额钻石投入，玩家也能稳定推进主要成长线。

#### 7.4.2 钻石
- `钻石` 是局外稀有万能货币，几乎可用于所有局外系统，但不应全面替代刷关价值。
- 钻石的主要用途包括：
  - 遗物抽卡
  - 购买稀有强化材料
  - 购买洗练保护或锁定类道具
  - 替代部分金币或素材缺口
  - 购买少量通用型强力装备
- 钻石主要由 `付费` 提供，但免费获取途径应比常规重付费手游略多，例如：
  - `首通`
  - `成就`
  - `挑战模式`
  - `签到`
  - `活动`

#### 7.4.3 货币设计原则
- `金币` 可以承担强化、普通资源购买和普通职业解锁，但 `不能进入高级卡池`。
- `钻石` 虽然万能，但最优用途应集中在 `遗物抽取` 与 `稀有资源补缺`，而不是全面覆盖所有系统。
- 应保证：
  - `金币保底可玩`
  - `钻石显著提速`
  - `刷关仍是装备毕业主来源`

### 7.5 获取与消耗闭环
- 日常基础循环应为：
  `刷关 -> 获得金币、装备、基础材料 -> 强化职业与装备 -> 挑战更高关卡`
- 长期追求循环应为：
  `积累钻石 -> 抽取遗物 / 购买稀有资源 -> 补强高端 build`
- 装备的主要来源固定为 `刷关掉落`，而不是商店直购。
- 遗物的主要来源固定为 `卡池抽取`，少量可通过高难活动、长期保底或特殊奖励获取。
- 重复装备可作为分解素材来源，重复遗物应转化为长期养成资源，避免重复获取完全无价值。

### 7.6 对局内的影响方式
- `职业` 负责影响：
  - 入局职业身份
  - 初始被动或职业特性
  - 流派推荐与 build 倾向
- `装备` 负责影响：
  - 入局基础属性
  - 特定流派的词条方向
  - 局内初始套装加成
- `遗物` 负责影响：
  - 常驻数值加成
  - 高稀有遗物提供的轻度机制增强
- 三者都只能作为 `局内构筑的起点和底盘`，不能替代局内通过 `技能强化 / 羁绊 / 宝物 / 烙印 / 成长武器` 形成的单局成型过程。

### 7.8 设计原则
- `职业` 决定玩法，不要让 `遗物` 反客为主。
- `装备` 是毕业主来源，不要让卡池通用装备替代刷图定向装备。
- `遗物` 是长期追求目标，但高稀有遗物的机制强化必须可控。
- `金币` 必须支撑大部分基础成长，保证非付费玩家也能稳定前进。
- `钻石` 必须有吸引力，但不能把所有系统压成单一最优解。
- 局外成长提升的是 `宽度`、`选择效率` 与 `终局上限`，而不是单纯的碾压数值。
- `职业模板档位` 只区分设计定位和资源投入，不等同于品质，也不直接写成玩家可见的稀有度。
