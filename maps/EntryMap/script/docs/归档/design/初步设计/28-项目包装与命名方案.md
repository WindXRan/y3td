# 项目包装与命名方案

> 用于统一项目的对外命名、平台包装、视觉气质与宣发口径。当前目标不是做“土味刷刷刷标题”，而是做一套既能让平台用户秒懂玩法、又有自己辨识度的产品包装。

## 1. 推荐定名

- 游戏正式名：`最后的防线`
- 平台展示名：`最后的防线：单英雄守线刷宝`
- 项目代号：`EntryMap`

命名理由：
- `最后的防线` 直接点出本作最核心的体验，就是单英雄死守最后一条线。
- 这个名字比生造词更容易懂，平台玩家第一眼就能理解题材和压力感。
- 名字自带“失守就结束”的末线氛围，和 `5 波 Boss 检定` 的结构天然契合。
- 展示名用副标题补足 `单英雄守线刷宝`，兼顾辨识度和平台点击转化。

## 2. 一句话包装

- 主宣传句：`一人守一线，五波定生死。`
- 扩展描述：`在敌潮压境的战场上，用宝物、羁绊、烙印和攻击技能构筑你的终盘。`

## 3. 产品包装定位

- 品类定位：`单英雄守线 + 局内构筑 + 刷宝成长 + 5 波 Boss 检定`
- 核心气质：`高压`、`硬朗`、`奇幻战场`、`稳重金属`、`战术桌面感`
- 体验关键词：
  `守线`
  `构筑`
  `刷宝`
  `成长`
  `Boss检定`
  `高风险挑战`

这套包装要强调的不是“无脑割草”，而是：
- 你在守一条真正会崩的战线
- 你每一轮选择都会改变 build 走向
- 你能在 20 分钟内明显感受到从普通到成型的战力跃迁

## 4. 对外卖点口径

- `单英雄守一线`：操作负担轻，但决策密度高。
- `五波定胜负`：单局节奏清楚，不拖沓。
- `局内构筑成型快`：攻击技能、羁绊、宝物、烙印层层叠强。
- `刷宝反馈直接`：每次强化、替换、成套都能马上看到输出变化。
- `挑战可主动加压`：贪更多资源，就承更多风险。

## 5. 视觉包装方向

### 5.1 主视觉关键词

- `怪潮压境`
- `最后防线`
- `孤英雄迎战`
- `暗金属 + 冷青能量 + 橙红警报`

### 5.2 主色建议

- 主底色：`铁黑`、`深灰`、`冷黑蓝`
- 主高亮：`暗金`、`冷青`
- 危险色：`橙红`
- 稀有奖励点缀：`蓝`、`紫`

### 5.3 封面构图建议

- 左侧前景放主英雄，形成“我方最后防线”的归属感。
- 右侧与远景放怪潮、精英和 Boss 压境，强调压迫感。
- 中间用技能、箭矢、雷火或冲击波把战场切成一条“战线”。
- 下方可少量露出 `宝物 / 羁绊 / 技能卡` 的构筑元素，但不能喧宾夺主。

## 6. 文案语气定义

- 语言要 `短`、`硬`、`有战场命令感`
- 少用搞笑口吻、兄弟口吻、轻浮网游腔
- 强调：
  `守住`
  `顶住`
  `成型`
  `翻盘`
  `防线`
  `Boss`

推荐用语示例：
- `怪潮来袭`
- `防线告急`
- `Build 成型`
- `宝物到手`
- `羁绊激活`
- `准备迎战 Boss`

不推荐用语示例：
- `兄弟集合`
- `开爽开刷`
- `爆了爆了`
- `无脑乱杀`

## 8. 游戏内包装统一口径

- 模式主标题建议统一叫：`标准守线`
- 挑战模式建议统一叫：`防线挑战`
- 构筑系统口径统一为：
  `攻击技能`
  `羁绊`
  `宝物`
  `烙印`
- 局内广播与 UI 应围绕：
  `防线`
  `怪潮`
  `构筑`
  `挑战`
  这四组词汇统一展开

## 9. Logo 方向建议

- 字形要厚重、横向展开，适合做 PC 端游式 Logo。
- `最后` 两字偏压迫、危险、末线感。
- `防线` 两字偏金属、秩序、稳固。
- 可加入轻微裂痕、锋刃、能量切线，但不要做成花哨手游发光字。

## 10. 备用名称

如果后续你觉得 `最后的防线` 还不够满意，优先可替换为以下备选：

- `战痕守线`
  更强调“死守”和战损感，风格更硬。
- `烙印战线`
  更强调 build 和奇幻构筑感，品牌气质更强。
- `孤线守望`
  更强调单英雄坚守，但商业感稍弱。

当前综合建议仍然优先使用：
- `最后的防线`

## 11. 当前结论

现阶段最适合本项目的平台包装方案是：

- 游戏正式名：`最后的防线`
- 平台展示名：`最后的防线：单英雄守线刷宝`
- 主宣传句：`一人守一线，五波定生死。`
- 包装核心：`怪潮压境 + 单英雄守线 + 局内构筑刷宝 + 硬朗奇幻战场`

这套命名和包装相比现有平台上的“刷刷刷”或“兄弟”类标题，更有自己的战场辨识度，同时又没有丢掉玩法可读性。

## 16. 首图构图草案

### 16.1 推荐首图结构

- 左侧前景：`主英雄`
- 中间：`技能爆发区 / 战线交锋区`
- 右侧中远景：`怪潮 + 精英 + Boss 压境`
- 上方：`游戏 Logo`
- 下方：`主标语 + 副标语 + 卖点短条`

这套结构的目标是：
- 第一眼看到“单英雄守线”
- 第二眼看到“怪潮和 Boss 压过来”
- 第三眼看到“刷宝构筑”的玩法信息

### 16.2 画面分区建议

#### A. 左侧主英雄区

- 英雄站位略靠左下或左中
- 姿态建议为 `迎战 / 蓄力 / 正在释放范围技能`
- 要有明显“独自扛线”的主角感
- 不建议做成多人站排或小队冲锋感

#### B. 中央战线交锋区

- 用箭矢、雷火、冲击波、冰霜或能量切线形成战场分割
- 明确出现“我方防线”与“敌方压境”的对冲关系
- 可以适度加入地面裂痕、技能轨迹、尸体轮廓或碎片增强压迫感
- 这里是整张图最适合表现“激烈战斗”的区域

#### C. 右侧怪潮区

- 普通怪形成压进阵列
- 前排可带 `高速怪 / 厚血怪 / 精英怪`
- 中后方建议放一只明显更大的 Boss 轮廓
- Boss 不一定要全身完整露出，但必须让玩家看出“后面还有更大的压力”

#### D. 文案信息区

- Logo 建议居上
- 主标语居中或居下偏中
- 副标语放在主标语下方
- 卖点短条放最底部或右下角

## 17. 平台详情页排版文案

### 17.1 详情页推荐结构

推荐从上到下采用：

1. `一句话卖点`
2. `核心玩法`
3. `为什么好玩`
4. `主要构筑系统`
5. `单局节奏`
6. `适合谁玩`

### 17.2 一句话卖点

- `单英雄守住最后的防线，在五波怪潮与 Boss 检定中刷宝构筑。`

### 17.3 核心玩法

- `你只控制一名英雄，独自守住整条战线。`
- `战斗以自动攻击为基础，重点不是拼手速，而是拼 build。`
- `每局固定 5 波，每波都有越来越高的怪潮压力和 Boss 检定。`
- `你需要在战斗中不断选择攻击技能、羁绊、宝物、烙印和成长武器路线。`

### 17.4 为什么好玩

- `上手快`：自动战斗降低操作门槛，前几分钟就能进入核心节奏。
- `成型快`：攻击技能、羁绊、宝物和烙印叠起来后，战力变化非常直接。
- `压力足`：怪潮、精英、Boss 和挑战事件会不断逼你做更贪或更稳的选择。
- `重开欲望强`：每一局 build 都不同，总会想试下一套更极端的终盘。 

### 17.5 主要构筑系统

#### 攻击技能

- `通过升级获得技能点，进入 3 选 1 攻击技能强化。`
- `你可以补新技能，也可以深挖已有技能路线。`
- `普攻、青木灵矢、赤炎箭、寒泉箭、乙木天雷等路线都能走出不同 build。`

#### 羁绊

- `消耗木材进行羁绊抽卡。`
- `同一羁绊成套后会获得额外强力效果。`
- `随着卡位吃满，你还需要做吞噬和替换决策。`

#### 宝物与烙印

- `宝物提供高价值局内强化。`
- `烙印提供更偏终盘和稀有的路线塑形。`
- `它们是拉开 build 上限和风格差异的关键。`

#### 成长武器

- `金币不会只拿来堆数字，还能持续强化成长武器。`
- `每 10 级词缀节点都会带来新的稳定成长方向。`

### 17.6 单局节奏

- `开局守线，快速建立第一轮输出。`
- `中期通过技能、羁绊和宝物逐渐成型。`
- `每波结束后迎战 Boss，检定当前 build 是否站得住。`
- `通关后还可以继续挑战更高压力内容。`

### 17.7 适合谁玩

- `喜欢魔兽 RPG 地图的玩家`
- `喜欢刷宝成长、数值变强、不断成型的玩家`
- `喜欢肉鸽 build，但不想被高强度操作拖累的玩家`
