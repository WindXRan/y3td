# 宝物旧版实现映射归档

> 本文归档旧版战斗内宝物 ID、效果组与推荐标签方案。以下内容不是当前实现事实，不可直接作为现行程序接入依据。

## 1. 文档目标

- 归档旧版宝物实现映射思路。
- 保留旧版 `treasure_id -> effect_group_id` 对照关系，便于追溯历史设计。
- 将旧方案与现行 `ITEM_###` 体系彻底隔离。

## 2. 当前事实

- 当前战斗内宝物事实源：`maps/EntryMap/script/data_csv/treasures.csv`
- 当前运行时入口：
  - `data/object_tables/treasure_catalog.lua`
  - `data/object_tables/treasure_catalog_compat.lua`
- 当前不再使用下列旧版 `treasure_id`
