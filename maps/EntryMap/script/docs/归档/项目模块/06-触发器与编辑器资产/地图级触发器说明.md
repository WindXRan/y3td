# 地图级触发器说明

## 模块职责

说明 `maps/EntryMap/global_trigger` 当前承担什么、不承担什么，避免 agent 在错误层面找主逻辑。

## Source Of Truth

- 地图级触发器：`maps/EntryMap/global_trigger`
- 运行时主逻辑：`maps/EntryMap/script`

## 当前目录结构

- `maps/EntryMap/global_trigger/trigger/UI`
- `maps/EntryMap/global_trigger/function/ui`
- 其余为索引文件与编辑器自动生成结构

## 关键状态与数据流

- 当前这里保存的是地图级 UI 触发器资产，不保存 `entry_runtime.lua` 的主状态
- Lua 主循环可以消费这里提供的 UI 资产，但当前没有把主逻辑搬到触发器里

## 当前已实现行为

- 地图级触发器主要承载 UI 相关编辑器资产
- 已存在的函数资产包括：
  - 选择单位/物品
  - 刷新单位属性
  - 刷新 Tooltip
  - 打开商店
  - 绑定物品栏和技能按钮

## 未实现或占位项

- 没有把挑战追踪、中央决策面板、结算面板正式落到触发器与 UI 交互链

## Agent 修改建议与风险点

- 除非目标明确是“接 UI 交互”或“复用编辑器现成函数”，否则不要优先改这里
- 改这里时要特别注意它和 Lua 运行时的边界，避免双向保存状态
