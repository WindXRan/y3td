# 链式羁绊内容重做 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在保留当前链路解锁、抽取、预览、进度展示机制的前提下，把现有链式羁绊内容重做为 `成长 / 暴击 / 箭术 / 法术 / 经济 / 体术` 六条主链，并让运行时文案、物编名称和编辑器技能描述全部对齐。

**Tech Stack:** Lua 5.4、Y3 运行时脚本、PowerShell、`py -3`、`luac`、`maps/EntryMap/script/tools/test_runtime_bonds_chain_smoke.py`、编辑器物编热更。
