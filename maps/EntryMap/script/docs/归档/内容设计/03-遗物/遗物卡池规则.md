# 遗物卡池规则

> 本文定义遗物卡池的获取结构和保底框架，不写具体概率。

## 1. 获取定位

- 遗物主要通过 `卡池抽取` 获取。
- 卡池是钻石侧最核心的长期追求入口之一。

## 2. 池子类型建议

| 池子类型 | 职责 | 说明 |
| --- | --- | --- |
| `常驻池` | 基础长期获取 | 主体遗物长期可得 |
| `限定池` | 主题投放 | 阶段性投放特定风格遗物 |
| `活动池` | 活动追求 | 用于活动或版本节奏补充 |

## 3. 保底原则

- 保底系统用于避免长期无高价值产出。
- 保底应优先保护：
  - `高品质出现`
  - `重复获取后的长期收益感`
- 保底不应直接把所有高稀有遗物做成无成本获取。

## 4. 货币规则

- `金币不能进入高级卡池`
- `钻石` 是主要抽取货币
- 免费钻石投放应明显存在，但整体获取速度慢于付费路径

## 5. 设计边界

- 卡池要提供长期追求感，但不能成为基础游玩门槛。
- 低投入玩家也应能稳定接触遗物系统。
- 高投入玩家应获得更快进度，而不是跳过全部过程。
