# 遗物卡池规则

> 本文定义遗物卡池的获取结构和保底框架，不写具体概率。

## 1. 获取定位

- 遗物主要通过 `卡池抽取` 获取。
- 卡池是钻石侧最核心的长期追求入口之一。

## 2. 池子类型建议

| 池子类型 | 职责 | 说明 |
| --- | --- | --- |
| `常驻池` | 基础长期获取 | 主体遗物长期可得 |
| `限定池` | 主题投放 | 阶段性投放特定风格遗物 |
| `活动池` | 活动追求 | 用于活动或版本节奏补充 |

## 3. 保底原则

- 保底系统用于避免长期无高价值产出。
- 保底应优先保护：
  - `高品质出现`
  - `重复获取后的长期收益感`
- 保底不应直接把所有高稀有遗物做成无成本获取。

## 4. 当前夺宝概率公示

> 以下内容按当前游戏内公示文案整理，作为抽奖系统实现与 UI 文案的同步依据。

### 4.1 基础概率与重复返还

| 品质 | 概率 | 重复返还 |
| --- | --- | --- |
| `R` | `80%` | 重复返还 `10 积分` |
| `SR` | `13%` | 重复返还 `50 积分` |
| `SSR` | `5%` | 重复返还 `120 积分` |
| `UR` | `2%` | 重复返还 `400 积分` |

### 4.2 首次抽奖保底

- 首次单抽：必定出现宝箱内 `第三个品质` 的道具。
- 首次十连：必定出现宝箱内 `第四个品质` 的道具。

### 4.3 后续抽奖保底

- 每累计 `60 抽`，必定出现宝箱内 `第四个品质` 的道具。

### 4.4 实现备注

- 当前公示文案使用的是“`第三个品质` / `第四个品质`”表述，落地时应和实际奖池排序保持一致。
- 若按当前公示顺序 `R -> SR -> SSR -> UR` 推断：
  - `第三个品质` 对应 `SSR`
  - `第四个品质` 对应 `UR`
- 如果后续奖池排序或命名调整，必须同步更新本页和 UI 概率公示。

## 5. 货币规则

- `金币不能进入高级卡池`
- `钻石` 是主要抽取货币
- 免费钻石投放应明显存在，但整体获取速度慢于付费路径

## 6. 设计边界

- 卡池要提供长期追求感，但不能成为基础游玩门槛。
- 低投入玩家也应能稳定接触遗物系统。
- 高投入玩家应获得更快进度，而不是跳过全部过程。
