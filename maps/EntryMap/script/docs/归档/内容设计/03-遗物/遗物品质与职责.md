# 遗物品质与职责

> 本文定义不同品质遗物在设计上的职责分工。

## 1. 品质体系

- 遗物统一采用六档品质：
  - `普通（白）`
  - `优良（绿）`
  - `精良（蓝）`
  - `史诗（紫）`
  - `传奇（橙）`
  - `传说（金）`

## 2. 职责分工

### 2.1 紫色及以下
- 只提供 `数值加成`
- 主要职责：
  - 平滑成长
  - 基础战力补强
  - 低门槛 build 适配

### 2.2 橙色
- 提供 `较强数值 + 轻度机制强化`
- 主要职责：
  - 成为中高端追求目标
  - 给 build 带来有限但明确的风格加成

### 2.3 金色
- 提供 `更强数值 + 更强但可控的机制强化`
- 主要职责：
  - 成为长期顶级追求目标
  - 强化已有玩法上限

## 3. 设计边界

- 橙金遗物不能单件改写一个职业的玩法定位。
- 低稀有遗物不能被做成毫无价值的填充物。
- 高稀有遗物要有追求感，但不能压过职业和装备系统的存在感。
