# 敌人与Boss总表

> 本文用于管理章节中的普通怪、精英怪与 Boss 内容池。

## 1. 文档职责

- 记录怪物主题、波次定位、机制标签与章节分布。
- 记录 Boss 的核心机制、检定目标与章节主题表现。
- 和 `docs/design/初步设计/05-关卡与波次设计.md` 配套使用：
  - `关卡与波次设计` 管节奏规则
  - `敌人与Boss总表` 管具体内容条目

## 2. 建议记录字段

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| `enemy_id` | 敌人 ID | 普通怪、精英、Boss 共用规则 |
| `display_name` | 名称 | 玩家可见名称 |
| `enemy_type` | 类型 | `普通怪 / 精英 / Boss` |
| `chapter_tags` | 章节标签 | 所属章节与主题 |
| `role_tags` | 机制定位 | 例如 `高速`、`厚血`、`后排`、`召唤` |
| `core_mechanics` | 核心机制 | 例如冲锋、分裂、召怪、减速 |
| `counter_goal` | 检定目标 | 要求玩家补什么能力 |
| `spawn_rule` | 出现规则 | 波次、时间轴、特殊阶段 |
| `status` | 当前状态 | `设计中 / 未实现 / 部分实现 / 已实现` |

## 3. 维护建议

- 敌人与 Boss 的内容设计要服务于：
  - 章节主题差异
  - build 检定差异
  - 5 波节奏中的前中后期压力变化
- 后续可继续拆出：
  - `普通怪总表.md`
  - `精英总表.md`
  - `Boss总表.md`
