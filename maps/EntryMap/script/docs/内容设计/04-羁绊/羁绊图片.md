# 羁绊图片

> 本文为当前羁绊设计主文档。
> 羁绊内容、命名、单卡属性、套装效果、UI期望表现一律优先以本文图片为准。
> 除图片中已经明确出现的内容外，其他额外补出的 OCR、文字恢复、推测说明都视为不可靠内容。
> `羁绊总表.md` 与 `链式羁绊内容重做方案.md` 只能作为实现映射记录，不得新增、改写或脑补图片里没有的设计。

## 1. 使用口径

- 图片里已经明确的卡名、数值、效果描述，不再被其他文字文档覆盖。
- 图片里没有明确出现的卡名、数值、效果描述，一律不算有效设计。
- `链式羁绊内容重做方案.md` 只负责把图片内容映射到当前链式节点结构。
- xmind 图只负责链路拓扑与前后置关系，不单独决定羁绊语义。
