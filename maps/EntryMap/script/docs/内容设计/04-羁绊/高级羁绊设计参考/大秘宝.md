#### **第二部分：SSR 传奇装备合成详表**

**传奇装备通用结构：**
1. **图纸（SSR）**：使用后触发对应“XX套装效果”，直接置入一个核心部件，并将剩余合成组件加入卡池。  
2. **核心部件A（SSR）**：图纸直接给予。  
3. **核心部件B（SSR）**：需先合成其下级 SR 形态，通常需要集齐3张 SR 组件卡。  
4. **合成公式**：`核心部件A（SSR） + 核心部件B（SSR） = 传奇装备成品（SSR）`  
5. **传奇装备强化**：成品可被 **UR海上传说** 吞噬，以强化其技能。

**总结**：
- 全卡组依旧保留高级羁绊的长线构筑感，玩家体验核心与“刀刀”一致。
- “启航套装”负责开流派，“见习水手 -> 海上传说”负责长线成长，“SSR传奇装备”负责中期构筑分叉，“UR终局装备”负责最终毕业。
- 包装重点是“热血冒险、寻宝、海上传说”这一层体验，适合做成带一点经典少年漫和海贼冒险味道的主题高级羁绊。
