#### **第二部分：SSR神兵合成详表**

**神兵通用结构**：

1. **图纸 (SSR)**：使用后触发对应“XX套装效果”，直接置入一个核心部件，并将剩余合成组件加入卡池。
    
2. **核心部件A (SSR)**：图纸直接给予。
    
3. **核心部件B (SSR)**：需先合成其下级SR形态，需集齐3张特定SR卡。
    
4. **合成公式**：`核心部件A (SSR) + 核心部件B (SSR) = 神兵成品 (SSR)`
    
5. **神兵强化**：成品可被 **UR刀刀大成**​ 吞噬，以强化其技能。
