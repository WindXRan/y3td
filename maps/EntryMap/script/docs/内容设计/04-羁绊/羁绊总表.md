## 3. 2026-04 分层规则

- `T1 / T2 / T3` 是目标设计层级，不等于当前 runtime 已实现品质。
- 进入进化链的羁绊必须满足“升阶同时升品”：
  - `T1` 子羁绊或素材羁绊建议目标品质为 `rare`
  - `T2` 上位羁绊建议目标品质为 `epic`
  - `T3` 母羁绊建议目标品质为 `legendary`
- 当前 runtime 已实现的 `common / rare / epic` 混合品质不需要在这轮文档中强行重写，只需要在设计层明确“未来进化链必须升品”。
- `result_type` 约定：
  - `terminal_swallow`：完成后吞噬释放槽位
  - `evolve_keep`：完成后留场，占槽，继续参与更高阶配方

## 5. 维护建议

- `当前 runtime 羁绊盘点` 只记录事实，不替代目标设计。
- `主推路线羁绊池` 记录的是目标内容池，可先写 `已实现 / 设计中` 两类状态。
- 后续若新增上位羁绊，必须同时写：
  - `tier`
  - `target_quality`
  - `bond_role`
  - `result_type`
  - `compose_inputs`
