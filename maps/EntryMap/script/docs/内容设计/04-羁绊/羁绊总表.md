## 3. 套装汇总

### 3.1 成长

套装效果：每秒木材 + 0.2，杀敌力量 + 0.1，杀敌敏捷 + 0.1，杀敌智力 + 0.1

- 智力提升：每秒智力 + 0.5
- 力量提升：每秒力量 + 0.5
- 敏捷提升：每秒敏捷 + 0.5
- 攻击提升：每秒攻击 + 0.5

### 3.2 经济

套装效果：每秒木材 + 0.4，每秒金币 + 10

- 金币多多：杀敌金币 + 10%
- 经验多多：杀敌经验 + 10%
- 杀敌多多：杀敌加成 + 10%

### 3.3 贪婪

套装效果：随机吞噬 4 张可吞噬卡牌

- 贪欲金币：金币 + 6000；放弃后直接除外，不返回卡池
- 贪欲之刃：杀敌数 + 888；放弃后直接除外，不返回卡池
- 贪欲钥匙：木材 + 150；放弃后直接除外，不返回卡池

### 3.4 暴击

套装效果：物理暴击 + 4%，魔法暴击 + 4%

- 暴击强化：物理暴击 + 3%，魔法暴击 + 3%
- 暴伤强化：物理暴伤 + 10%，魔法暴伤 + 10%

### 3.5 法神
