# 通用卡池装备规则

> 本文定义卡池中通用型装备的定位边界，不填写具体装备条目。

## 1. 系统定位

- 通用卡池装备用于：
  - 补进度
  - 补泛用战力
  - 缓解非毕业期的装备断层
- 通用卡池装备不负责替代刷图毕业装备。

## 2. 强度边界

- 同品质下，通用装备对特定流派的适配度应弱于刷图定向装备约 `20%`。
- 通用装备更适合提供：
  - 平均化属性
  - 泛用套装
  - 更低门槛的过渡选择

## 3. 设计原则

- 通用装备要“够用”，但不能“最优”。
- 高投入玩家可以靠它加速，但最终毕业仍应回到刷图体系。
- 通用装备最适合承担：
  - 新职业过渡
  - 多职业共用
  - 前中期平滑体验
