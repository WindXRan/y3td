# 星纹行者

> `星纹行者` 是首发职业池中的 `高上限模板` 职业，核心卖点是连锁、切换和多状态循环。

## 1. 职业定位

- 模板档位：`高上限模板`
- 玩家标签：`高上限`
- 核心爽点：
  - 切状态
  - 打连锁
  - 抓窗口
  - 成型后最花、最能整活
- 适合人群：
  - 熟练玩家
  - 喜欢研究 build 的玩家
  - 愿意投入更多资源换取高上限的玩家

## 2. 核心机制

- 以 `多状态切换`、`连锁触发`、`节奏循环`、`姿态收益` 为核心。
- 星纹行者不是“拿到就强”的职业，而是：
  - 需要先理解状态之间的关系
  - 再理解什么时候切、为什么切
  - 最后通过循环把收益打满
- 这个职业的乐趣来自：
  - 操作思路上的整活
  - build 组合上的整活
  - 资源投入后的明显回报

## 3. 成长曲线

### 3.1 前期
- 强度不一定突出。
- 玩家需要先熟悉状态切换和循环节奏。

### 3.2 中期
- 状态之间开始相互喂收益后，职业手感会明显变顺。
- 一旦形成稳定轮转，强度提升会很明显。

### 3.3 后期
- 成型后是首发池里上限最高、变化最多的职业。
- 适合承担“熟练玩家整活”的终盘角色。

## 4. 流派分支

### 4.1 星火连锁流
- 关键词：
  - `连续触发`
  - `连段收益`
  - `爆发窗口`
- 玩法重点：
  - 依靠连续触发把整套输出连起来
  - 更强调节奏感和触发密度

### 4.2 双相切换流
- 关键词：
  - `姿态切换`
  - `爆发窗口`
  - `状态轮转`
- 玩法重点：
  - 在两种核心状态之间切换
  - 不同状态承担不同任务，例如清怪和爆发

### 4.3 印记循环流
- 关键词：
  - `挂印记`
  - `引爆`
  - `循环收益`
- 玩法重点：
  - 先挂印记，再通过切换和触发引爆
  - 更强调 build 理解和完整循环

## 5. 天赋树方向

### 5.1 基础层
- 状态持续时间
- 切换效率
- 资源恢复
- 循环稳定性

### 5.2 分支层
- `连锁强化`
  - 高频触发收益
- `切换强化`
  - 状态轮转效率
- `印记强化`
  - 印记生成和引爆收益

### 5.3 核心层
- 完整循环后的质变奖励
- 多状态协同收益
- 高投入后的终盘上限释放

## 6. 推荐标签

### 6.1 推荐装备标签
- `技能循环`
- `触发收益`
- `冷却`
- `资源回复`

### 6.2 推荐遗物标签
- `连锁强化`
- `状态切换`
- `多段收益`

## 7. 设计边界

- 不能低投入就全面压过主力模板。
- 难点和高成本必须真实存在。
- 回报也必须真实更高，才能支撑它作为高阶职业的定位。
- 如果它在低练度也万能，则会破坏首发职业池的成本与定位分层。
