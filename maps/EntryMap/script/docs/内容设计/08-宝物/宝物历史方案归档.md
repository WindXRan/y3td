# 宝物历史方案归档

> 本文仅用于保留旧版战斗内宝物设计思路。以下内容不是当前实现，不可直接作为程序、配表或 UI 的现行接入依据。

## 1. 文档目标

- 归档旧版宝物分层方案。
- 保留未接入现行图鉴的历史临时宝物原型。
- 避免这些内容继续混在主文档里被误读为当前事实。

## 2. 作用范围

- 包含：
  - 旧版 `general / route_amp / tactical_temp` 分层概念
  - 未接入现行 `treasures.csv` 的历史临时宝物原型
- 不包含：
  - 当前图鉴宝物事实
  - 当前运行时兼容层规则

## 3. 历史分层定义

| 类型 | 历史含义 | 当前状态 |
| --- | --- | --- |
| `general` | 泛用常驻强化 | `已废弃，仅保留概念` |
| `route_amp` | 特定构筑放大件 | `已废弃，仅保留概念` |
| `tactical_temp` | 波次/挑战/Boss 临时强化 | `已废弃，仅保留概念` |

## 5. 使用约定

- 如需查看当前战斗内宝物，请返回 [宝物总表](./宝物总表.md)。
- 如需查看当前投放口径，请返回 [宝物分层与投放规则](./宝物分层与投放规则.md)。
- 若未来要复活这些历史原型，应按现行 `ITEM_###` 体系重新立项，而不是直接恢复旧 `treasure_id`。
