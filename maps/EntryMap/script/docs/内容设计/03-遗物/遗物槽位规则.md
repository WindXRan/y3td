# 遗物槽位规则

> 本文定义遗物槽位结构，不填写具体遗物条目。

## 1. 当前定案

- 遗物采用 `少量固定槽位`，避免数量过多后压过职业与装备系统。
- 槽位数量应以“能形成长线追求，但不会变成无限堆叠”为原则。

## 2. 槽位职责建议

| 槽位 | 职责 | 说明 |
| --- | --- | --- |
| `核心槽` | 主遗物位 | 用于承载最核心的高价值遗物 |
| `辅助槽 1` | 数值补强 | 更偏平滑数值成长 |
| `辅助槽 2` | 流派补强 | 更偏职业或 build 适配 |

## 3. 槽位设计原则

- `核心槽` 更适合放高稀有遗物。
- `辅助槽` 更适合承担平滑成长和方向修正。
- 不应通过无限加槽位来解决长期追求问题。
