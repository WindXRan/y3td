# 技能总表

> 本表分为两层：
> 1. `当前已实现` 的 `5` 个攻击技能，事实来源以 `maps/EntryMap/script/entry_objects/attack_skills` 为准。
> 2. `已入库未实装` 的第二批通用攻击技能，事实来源以 `maps/EntryMap/script/entry_objects/attack_skill_blueprints/second_batch_skills.lua` 与对应设计文档为准。

## 1. 文档职责

- 记录五个基础攻击技能的当前事实。
- 记录第二批通用攻击技能的工程内设计资产落点。
- 记录每个技能的核心身份、中期质变点、终盘质变点与适配路线。
- 不直接把未实装技能伪装成已接入 runtime 的真实技能。
- 第二批技能的强化结构统一采用 `普通 / 优秀 / 稀有 / 传奇` 四品质与 `9+1` 模板。

## 2. 建议记录字段

| 字段 | 含义 |
| --- | --- |
| `skill_id` | 技能 ID |
| `display_name` | 技能名称 |
| `current_summary` | 当前 runtime 描述 |
| `damage_type` | 当前伤害类型 |
| `base_damage_ratio` | 当前基础伤害倍率 |
| `base_cooldown` | 当前基础冷却 |
| `base_range` | 当前基础射程 |
| `core_identity` | 目标核心身份 |
| `midgame_spike` | 中期质变点 |
| `endgame_spike` | 终盘质变点 |
| `best_routes` | 适配路线 |
| `best_bonds` | 适配羁绊 |
| `best_treasures` | 适配宝物 |
| `status` | 当前状态 |
