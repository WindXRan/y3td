# 技能总表

> 本表只覆盖当前已实现的 `5` 个基础攻击技能；事实来源以 `maps/EntryMap/script/entry_objects/attack_skills` 为准，目标定位以 2026-04 局内体验内容深化设计为准。

## 1. 文档职责

- 记录五个基础攻击技能的当前事实。
- 记录每个技能的核心身份、中期质变点、终盘质变点与适配路线。
- 不在本表中提前扩展新的基础技能位。

## 2. 建议记录字段

| 字段 | 含义 |
| --- | --- |
| `skill_id` | 技能 ID |
| `display_name` | 技能名称 |
| `current_summary` | 当前 runtime 描述 |
| `damage_type` | 当前伤害类型 |
| `base_damage_ratio` | 当前基础伤害倍率 |
| `base_cooldown` | 当前基础冷却 |
| `base_range` | 当前基础射程 |
| `core_identity` | 目标核心身份 |
| `midgame_spike` | 中期质变点 |
| `endgame_spike` | 终盘质变点 |
| `best_routes` | 适配路线 |
| `best_bonds` | 适配羁绊 |
| `best_treasures` | 适配宝物 |
| `status` | 当前状态 |
