# 结论与当前定案

> 从《项目主 GDD（初版）》拆分而来，便于按专题维护。

## 16. 当前版本结论

本项目的核心方向可以定义为：

“以单英雄驻守战线为表现形式，以自动战斗降低操作负担，以攻击技能强化、羁绊抽卡、装备与烙印构筑为主要决策内容，以固定 5 波推进、每波 240 秒怪物阶段和每波 Boss 检定为主节奏，并在通关后以挑战模式承接局外存档奖励的 RPG 肉鸽地图。”

后续脚本、物编、UI 与数值设计，都应优先围绕这个闭环展开。


## 17. 当前定案

- 标准模式固定为 `5 波`。
- 每波怪物阶段固定 `240 秒`。
- 每波只配置 `1 种主怪` 与 `1 名 Boss`。
- 击败第 `5 波 Boss` 后立即胜利。
- 通关后可进入挑战模式，通过挑战词条获取写入局外存档的奖励。


## 18. 当前包装定案

- 对外包装名暂定为 `最后的防线`。
- 平台展示名建议为 `最后的防线：单英雄守线刷宝`。
- 主宣传句定为 `一人守一线，五波定生死。`
- 包装方向统一为：`怪潮压境`、`单英雄守线`、`局内构筑刷宝`、`硬朗奇幻战场`。

## 19. 局外成长当前定案

- 局外长期成长固定为 `职业`、`装备`、`遗物` 三条主线。
- `职业` 是核心玩法系统，采用 `解锁 + 养成 + 天赋树` 结构，但 `不使用品质分级`。
- `职业` 内部采用 `入门模板 / 主力模板 / 高上限模板` 三档设计定位：
  - `入门模板` 负责新玩家快速上手
  - `主力模板` 是平民玩家主力，综合性价比最高
  - `高上限模板` 面向熟练玩家，机制更特殊，所需金币和养成材料更多
- `装备` 有品质，主要靠 `刷关掉落` 获得，采用 `随机词条 + 洗练 + 套装效果` 结构，并随关卡推进逐步抬升品质。
- `遗物` 主要靠 `卡池抽取` 获得：
  - `紫色及以下` 只提供数值加成
  - `橙色` 提供轻度机制加成
  - `金色` 提供更强但可控的机制加成
- 局外货币固定为：
  - `金币`：强化、普通资源购买、普通职业解锁
  - `钻石`：万能货币，可参与大多数局外行为
- `金币不能进入高级卡池`；卡池中的通用型装备只用于补进度和泛用战力，同品质下应弱于刷图定向装备。
