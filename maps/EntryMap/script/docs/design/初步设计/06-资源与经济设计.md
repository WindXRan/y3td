# 资源与经济设计

> 从《项目主 GDD（初版）》拆分而来，便于按专题维护。

## 10. 资源与经济设计

### 10.1 局内资源
- 首版局内正式启用的资源固定为：
  `经验`
  `技能点`
  `金币`
  `木材`
  `挑战次数`
- `临时能量资源` 暂不进入首版正式循环；若后续要加入章节机关、主动大招或地图技能，再单独扩展，不与当前 5 波 MVP 经济混用。

#### 10.1.1 经验与技能点
- `经验` 用于提升英雄等级。
- 英雄每升 `1 级`，固定获得 `1 个技能点`。
- `技能点` 不会自动消耗，允许玩家手动囤积，等到合适时机再按 `G` 连续处理多次强化。
- `技能点` 只对应 `攻击技能强化 3 选 1`，不参与羁绊、宝物、成长武器或挑战事件的成本结算。

#### 10.1.2 金币
- `金币` 是首版最稳定的战斗资源，主要职责是推动 `成长武器等级`。
- 金币默认不用于 `G`、`F`、宝物候选与挑战事件，不和其他成长入口争抢同一货币。
- 金币的主要出口固定为 `成长武器强化 +1`；首版不额外加入购买消耗品、波间商店或装备倒卖。

#### 10.1.3 木材
- `木材` 是首版第二条核心成长货币，专门服务于 `羁绊抽卡`。
- 每次按 `F` 或点击 `羁绊抽卡` 按钮时，固定消耗 `100 木材` 生成一轮候选。
- 羁绊面板的手动刷新也消耗木材，因此木材承担的是“拼路线”和“修正路线”的双重职责。

#### 10.1.4 挑战次数
- `挑战次数` 是事件系统专用资源，与 `金币 / 木材 / 技能点` 完全分离。
- 首版建议：
  `上限 = 3`
  `开局初始 = 1`
  `每 120 秒恢复 1 次`
- 挑战次数不写入局外存档，也不允许通过金币、木材或其他货币直接购买。

### 10.2 经济决策点
- 立即提升战力，还是暂时保留资源等关键节点。
- 先补稳定成长，还是追一条更贪的构筑路线。
- 当前波次更需要 `清怪`、`站场`、`追等级` 还是 `抢事件奖励`。

#### 10.2.1 金币决策
- 首版不做 `利息` 系统，因此金币不存在“存钱吃息”的长期收益逻辑。
- 金币的核心决策变为：
  `现在立刻升成长武器补稳态战力`
  或
  `先攒到下一个 10 级词缀节点，一次完成质变`
- 玩家通常要在“马上加几点基础强度”和“留金币冲下个词缀节点”之间取舍。

#### 10.2.2 木材决策
- 木材的核心决策是：
  `现在开 F 补卡`
  或
  `继续攒木材，等待更合适的抽卡窗口`
- 因为羁绊抽卡附带刷新费用，木材不只是“开门票”，还是“修候选”的后续预算。
- 当玩家已经接近某个关键羁绊阈值时，木材价值会明显上升；此时木材挑战的优先级也应随之提高。

#### 10.2.3 技能点决策
- 技能点不像经典升级奖励那样必须当场立刻点掉，首版允许玩家积攒后集中使用。
- 技能点的核心决策是：
  `立刻把当前强势技能做深`
  或
  `先等新技能 / 新路线出现后再集中投入`
- 这让玩家在第 `2 到 4 波` 期间能够做出“先拿广度还是先做深度”的清晰分歧。

#### 10.2.4 挑战次数决策
- 挑战次数的核心决策是：
  `现在开低风险稳定补给`
  或
  `留着等宝物挑战 / 关键时段再开`
- 因为挑战事件与主线、Boss 同场并行，挑战次数实际上也是一种“主动加压换资源”的风险预算。

### 10.3 首版资源来源
首版建议按“主线稳定来源 + 支线爆发来源”理解资源来源：

| 资源 | 主线稳定来源 | 支线爆发来源 | 主要用途 |
| --- | --- | --- | --- |
| `经验` | 普通怪击杀、精英/Boss 击杀 | `经验挑战` | 升级与获得技能点 |
| `技能点` | 英雄升级 | 无 | `G` 攻击技能强化 |
| `金币` | 普通怪击杀、精英/Boss 奖励、部分宝物/羁绊效果 | `金币挑战` | 成长武器强化 |
| `木材` | 精英/Boss 奖励、特殊高价值掉落、部分羁绊效果 | `木材挑战` | `F` 羁绊抽卡与刷新 |
| `挑战次数` | 时间恢复 | 无 | 开启四类挑战事件 |

补充约定：
- 普通怪默认主要掉 `经验 + 金币`，不把木材做成普通小怪的大宗日常掉落。
- `木材` 应更多来自 `精英 / Boss / 事件奖励 / 羁绊效果`，这样它能稳定维持“高价值成长货币”的体感。
- `宝物` 不作为普通货币出现，而是通过 `宝物挑战` 或高价值奖励面板直接产出。

### 10.4 首版资源消耗与费用规则
首版所有主要消费动作建议固定如下：

| 消费动作 | 消耗资源 | 首版费用规则 | 备注 |
| --- | --- | --- | --- |
| `攻击技能强化 3 选 1` | `技能点` | `1 技能点 / 次` | 不支持金币代替 |
| `羁绊抽卡` | `木材` | `100 木材 / 次` | 开启一轮候选 |
| `羁绊抽卡刷新` | `木材` | `第 1 次免费`，后续 `20 / 40 / 60` 木材 | 第 `4` 次及以后固定 `60` |
| `成长武器强化 +1` | `金币` | 按等级区间递增 | 详见 `6.2.1A` |
| `词缀节点刷新` | 无 | `3 次免费` | 首版不开放额外付费刷新 |
| `四类挑战事件` | `挑战次数` | `1 次 / 次` | 四类挑战共用次数池 |
| `宝物候选` | 无 | `0` | 成本已包含在奖励来源内 |

### 10.5 经济节奏目标
首版资源节奏建议围绕 `前期开路 -> 中期分流 -> 后期检定` 三段展开：

| 时段 | 资源节奏目标 | 推荐玩家感受 |
| --- | --- | --- |
| `第 1 波` | 先建立 `金币 -> 成长武器` 的稳定反馈，并尽快完成第一次 `G` 或 `F` 选择 | “我能立刻变强，而且每种资源都有去处” |
| `第 2 波` | 开始让玩家在 `补技能`、`抽羁绊`、`攒金币冲词缀` 之间做第一次真实取舍 | “资源开始不够同时做所有事了” |
| `第 3 波` | 挑战事件开始明显影响路线成型速度，资源选择开始出现贪与稳的差异 | “这一把开始像 build 了” |
| `第 4 波` | 金币与木材都进入紧张期，玩家需要根据当前短板决定优先补哪条线 | “该把资源压在哪个系统上最值” |
| `第 5 波` | 大部分日常消费应已完成，资源更偏向做终盘补强与收官挑战 | “我是在冲通关还是冲更高收益” |

### 10.6 成长武器金币曲线目标
首版建议把成长武器视为整局金币经济的主轴，并以如下节点作为调数参考：
- `第 1 波结束前`：武器达到约 `8 到 10 级`，玩家完成第一次词缀节点或接近该节点。
- `第 2 波结束前`：武器达到约 `16 到 20 级`，第二次词缀节点开始主导中期强度。
- `第 3 波结束前`：武器达到约 `24 到 30 级`，build 开始出现明显的普攻 / 技能分流。
- `第 4 波结束前`：武器达到约 `34 到 40 级`，成为通关前的稳定强度底盘。
- `第 5 波期间`：视挑战与宝物收益情况冲刺 `45 到 50 级`，承担终局检定功能。

若实机出现以下问题，则优先从金币曲线调整：
- 玩家第 `2 波` 之前就能轻松冲过 `20 级`
  说明金币过剩，成长武器中期节点过早被吃满。
- 玩家到第 `4 波` 还长期停留在 `20 多级`
  说明金币不足，成长武器没有承担起稳态成长支柱的职责。

### 10.7 防卡死与防膨胀规则
- 普通数值资源统一采用 `直接入账`，不通过地面掉落拾取，避免资源丢失和无意义走位成本。
- 技能点允许囤积，避免因为战斗高压期不方便点卡而白白浪费升级收益。
- 羁绊抽卡支持 `稍后选择`，避免玩家在 Boss 压力期被迫立刻做长决策。
- 高价值奖励若因已有中央面板阻塞而暂时不能弹出，必须写入 `待领取奖励` 队列，先保底逻辑入账。
- 首版不做 `金币利息`、`木材利息`、`资源衰减`、`付费买复活` 这类会显著扭曲节奏的经济规则。
- 首版也不做多货币汇率互换，例如 `金币换木材`、`木材换技能点`，避免玩家把经济问题转化为单一最优解。

### 10.8 资源设计总原则
- 每种核心资源尽量只服务 `1 套主要系统`，避免资源职责重叠。
- 让玩家在大多数时刻都知道“这笔资源下一步可以拿来做什么”。
- 让主线波次始终是资源底盘，让挑战事件成为额外爆发，而不是反过来。
- 让高价值奖励更多承担“路线改变”而不是“纯数值通胀”。
