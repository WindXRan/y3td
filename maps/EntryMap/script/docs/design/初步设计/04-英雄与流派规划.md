# 英雄与流派规划

> 从《项目主 GDD（初版）》拆分而来，便于按专题维护。

## 8. 英雄与流派规划

### 8.0 职业模板档位
- 职业系统虽然 `不使用品质分级`，但职业模板本身仍分为 3 类设计档位，用于控制上手门槛、投入成本与成长上限。
- 对玩家建议显示为功能标签，而不是品质标签，例如：
  - `易上手`
  - `推荐主力`
  - `高上限`

#### 8.0.1 入门模板
- 关键词：`直观`、`便宜`、`成型快`
- 目标用户：第一次接触本玩法的新玩家。
- 设计要求：
  - 机制尽量少绕弯，战斗反馈直接。
  - 低投入即可成型，适合作为初始职业或前期赠送职业。
  - 允许上限偏低，但不能弱到没有持续使用价值。

#### 8.0.2 主力模板
- 关键词：`中庸`、`稳定`、`强`
- 目标用户：已经理解玩法、准备长期投入的普通玩家。
- 设计要求：
  - 成本略高于入门模板，但性价比最高。
  - 机制不求最花，但强度要足、手感要稳。
  - 应成为大多数平民玩家的主力职业池。

#### 8.0.3 高上限模板
- 关键词：`特殊`、`整活`、`高投入`
- 目标用户：熟练玩家、高投入玩家、喜欢研究 build 的玩家。
- 设计要求：
  - 机制更特殊，允许存在更明显的 build 分歧和上限空间。
  - 解锁金币更多，养成材料也更多。
  - 低投入状态下不一定比主力模板强，但成型后上限最高。

### 8.1 首发英雄
- 战士型：稳健站场，依赖吸血、反击、近战范围清怪。
- 游侠型：高攻速高暴击，擅长穿透、弹射、连射。
- 法师型：技能驱动，依赖能量循环和范围爆发。

### 8.2 典型流派
- 普攻暴击流。
- 攻速叠层流。
- 技能循环流。
- 召唤物辅助流。
- 反伤 / 受击触发流。
- 冰冻 / 灼烧 / 中毒等异常状态流。

### 8.3 流派设计要求
- 每个英雄至少有 2 到 3 条可行 build 路线。
- 通用强化要能跨英雄复用，减少内容制作成本。
- 专属强化要保证英雄辨识度。
- 首发职业池建议同时覆盖：
  - 至少 `1 个入门模板`
  - 至少 `1 到 2 个主力模板`
  - 至少 `1 个高上限模板`
- 这样可以同时满足：
  - 新玩家快速上手
  - 普通玩家稳定养成
  - 熟练玩家研究高上限玩法
