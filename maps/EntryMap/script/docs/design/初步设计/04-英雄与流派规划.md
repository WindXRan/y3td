# 英雄与流派规划

> 从《项目主 GDD（初版）》拆分而来，便于按专题维护。

## 8. 英雄与流派规划

### 8.1 首发英雄
- 战士型：稳健站场，依赖吸血、反击、近战范围清怪。
- 游侠型：高攻速高暴击，擅长穿透、弹射、连射。
- 法师型：技能驱动，依赖能量循环和范围爆发。

### 8.2 典型流派
- 普攻暴击流。
- 攻速叠层流。
- 技能循环流。
- 召唤物辅助流。
- 反伤 / 受击触发流。
- 冰冻 / 灼烧 / 中毒等异常状态流。

### 8.3 流派设计要求
- 每个英雄至少有 2 到 3 条可行 build 路线。
- 通用强化要能跨英雄复用，减少内容制作成本。
- 专属强化要保证英雄辨识度。
