# 文字颜色显示规则

## 适用范围

- 当前首版先用于 `F` 羁绊三选一面板。
- 目标文件：
  - `maps/EntryMap/script/runtime/bonds_chain.lua`
  - `maps/EntryMap/script/runtime/choice_panel_model.lua`
  - `maps/EntryMap/script/ui/choice_panel.lua`
  - `maps/EntryMap/script/ui/choice_panel_text_colorizer.lua`

## 版式规则

- 羁绊标题固定显示为 `羁绊名 (当前/需求)`，例如 `体术 (1/3)`。
- 单卡名称单独一行显示在图标下方。
- 单卡基础效果显示在基础效果区。
- 套装说明固定拆为两段：
  - 标题：`激活[体术]套装效果：`
  - 正文：套装效果描述

## 默认颜色

- 羁绊标题：`bond_red`
- 单卡名称：`blue`
- 基础效果：`green`
- 套装标题：`gold`
- 套装正文默认底色：`dim`

## 默认字号

- 品质徽记：`18`
- 羁绊标题：`26`
- 单卡名称：`23`
- 基础效果：`24`
- 套装标题：`22`
- 套装正文：`18`

以上字号由 `maps/EntryMap/script/ui/choice_panel.lua` 统一下发，不再依赖 prefab 默认值。

## 自动染色规则

自动染色只作用于 `套装正文`，不改 `基础效果` 的默认绿色表现。

### 优先级

从高到低：

1. 关键词规则
2. 数值规则
3. 默认正文颜色

如果同一段文本同时命中多条规则，取更高优先级的颜色，不再继续覆盖。

### 关键词高亮

命中以下关键词时，文字显示为 `green`：

- `自适应伤害`
- `物理伤害`
- `魔法伤害`
- `所有伤害`
- `伤害减免`
- `攻击速度`
- `攻击力`
- `生命值`
- `技能急速`
- `力量`
- `敏捷`
- `智力`

### 数值高亮

命中以下模式时，文字显示为 `cyan`：

- 百分比：如 `60%`
- 普通数值：如 `100`、`1200`

## 新增规则的方式

在 `maps/EntryMap/script/ui/choice_panel_text_colorizer.lua` 里追加规则：

- 固定词：放进 `keywords`
- 模式词：放进 `patterns`
- 如果新规则要覆盖旧规则，给它更高优先级

## 当前约束

- 现阶段采用“正文底色 + 命中词覆盖高亮”的轻量方案，不依赖富文本控件。
- 自动染色目前只保证羁绊面板可用；后续如果宝物或技能卡要复用，沿用同一规则文件扩展即可。
