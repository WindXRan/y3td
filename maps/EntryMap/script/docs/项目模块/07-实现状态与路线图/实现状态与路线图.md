- 5 波主线流程

## 部分实现

- 地图有 `GameHUD`、运行时 HUD 与若干 UI 资产，但正式中央决策面板、背包、奖励记录页未落地

- 高价值决策目前主要通过文本消息与 HUD 按钮交互，不是完整卡面 UI

## Agent 修改建议与风险点

- 新系统进入实现前，先回答三个问题：
  - 状态放哪
  - 与现有 `G/F/烙印/宝物/挑战` 如何并存
  - UI 是读什么 runtime
- 不要直接按设计稿文件名开做；先确认当前代码里是否已有相邻状态和入口
