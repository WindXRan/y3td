# 羁绊最终表（职业包装版）

规则统一：
- 每套 3-5 张卡
- 允许 0-1 张功能卡
- 允许纯属性型
- 允许凑齐后才出功能
- 允许凑齐后仍只给属性
- 弓箭相关内容尽量转到普攻技能体系，不放在羁绊主体系里
