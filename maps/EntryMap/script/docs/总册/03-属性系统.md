# 英雄属性系统

## 文档定位

- 属性定义源：`script/runtime/hero_attr_system.lua`
- 属性是英雄战斗能力的核心量化指标
- 本文档以代码中注册的 `AttrDefs.list` 为准

---

## 属性面板展示顺序

属性面板按以下顺序显示主要属性：

```
攻击 -> 攻击白字 -> 攻击绿字 -> 攻击速度 -> 攻击范围 ->
生命 -> 护甲 -> 护甲白字 -> 护甲绿字 ->
力量 -> 力量白字 -> 力量绿字 ->
敏捷 -> 敏捷白字 -> 敏捷绿字 ->
智力 -> 智力白字 -> 智力绿字 ->
最终攻击 -> 最终生命 -> 最终护甲 ->
攻击结算值 -> 生命结算值 -> 护甲结算值
```

---

## 配置注意事项

1. **属性名必须已注册** - 使用的属性名必须在 `AttrDefs.list` 中定义
2. **百分比属性** - 使用小数格式 (0.1 表示 10%)
3. **同名属性可叠加** - 多个来源的同名属性会自动累加
4. **派生属性** - 带 `derived_output = true` 的属性由系统自动计算
