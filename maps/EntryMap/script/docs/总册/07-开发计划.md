4. 局外长期成长与写回
5. 多英雄与章节内容扩展

## 文档维护计划

1. 新需求优先写入本目录总册
2. 旧目录仅在需要追溯细节时查阅
3. 当总册稳定后，再考虑把旧目录整体转为 `archive`
