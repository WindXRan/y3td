# 玩法与系统设计

## 战斗结构

- 单英雄自动索敌与自动施法
- 固定 `5 波` 主线
- 每波包含主怪阶段与 Boss 阶段
- 挑战与主线并行，不切独立战场

## 局内成长

- `G`：攻击技能强化
- `F`：羁绊抽卡
- `宝物`：高价值构筑强化
- `烙印`：等级节点奖励
- `金币`：后续用于成长武器与词缀系统

## 攻击技能结构

- `1 个固定基础位 + 4 个自由攻击技能位`
- 第一批 5 技能已经接入 runtime
- 第二批技能已完成结构化设计与数据化入库，待接入正式施法与卡池
