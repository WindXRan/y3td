# 文档总册阅读入口

> 本目录是当前项目的唯一推荐文档入口。目标是用少量主文档覆盖 `设计 / 内容 / 实现 / 运行时 / UI / 计划` 六类信息，降低在多目录之间来回跳转的成本。

> 旧目录 `归档/design / 归档/内容设计 / 归档/项目模块 / 归档/开发进度与计划` 统一作为归档区，仅在需要追来源、看细节、查历史方案时进入。

## 文档目标

- 以最少主文档承接当前项目的可阅读信息。
- 让新读者先看总册，再按需回到旧目录查细节。

## 作用范围

- 包含：总览、系统设计、内容总表、实现映射、运行时结构、UI规范、开发计划。
- 不包含：历史版本逐条讨论、旧方案长链路拆分、模块级备忘录式细节。
