# 总览与术语

## 项目定位

- 项目代号：`EntryMap`
- 当前包装方向：`修仙 / 五行 / 单英雄守线刷宝`
- 核心体验：单英雄自动战斗、波次刷怪、局内构筑、局外成长

## 当前主循环

1. 局外选章节与模式
2. 进入单局战斗并自动守线
3. 击杀敌人获得经验、金币、木材
4. 通过 `G / F / 宝物 / 烙印` 持续构筑
5. 完成 `5 波 + Boss` 后胜负结算
6. 将结果回流到局外解锁与记录

## 当前世界观与包装口径

- 玩家可见语义统一采用 `修仙 / 五行`
- 不再使用旧的 `火冰电风 / 能量 / 物系` 作为主展示口径
- 当前没有 `五行相克`
- 当前没有敌方五行抗性系统

## 伤害字段口径

| 字段 | 用途 |
| --- | --- |
| `damage_type` | 旧兼容字段，保留 `物理 / 法术` |
| `damage_form` | 运行时形态，如 `weapon / spell / dot / summon` |
| `element` | 五行字段，如 `metal / wood / water / fire / earth / none` |
| `damage_label` | 玩家可见伤害名，如 `金行箭罡` |
