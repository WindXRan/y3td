# 平台宝箱抽奖规则

## 一、宝箱类型与定价

### 1.1 传说宝箱
- **单次抽取**：8元/次
- **十连抽取**：72元/次（享9折优惠）
- **奖池品质**：N/R/SR/SSR

### 1.2 神话宝箱
- **单次抽取**：12元/次
- **十连抽取**：108元/次（享9折优惠）
- **奖池品质**：R/SR/SSR/UR

## 二、单次抽奖概率

### 2.1 传说宝箱

| 品质 | 概率 |
|------|------|
| SSR | 1.5% |
| SR | 4% |
| R | 25% |
| N | 69.5% |

### 2.2 神话宝箱

| 品质 | 概率 |
|------|------|
| UR | 1% |
| SSR | 2% |
| SR | 8.2% |
| R | 88.8% |

## 三、十连抽概率

### 3.1 传说宝箱

每进行10次十连抽抽奖平均可获得：
- SSR道具：2个
- SR道具：11个
- R道具：20个
- N道具：67个

> **说明**：该数据为大量数据进行的平均测算，玩家实际获得的数据可能有所变化。

### 3.2 神话宝箱

每进行10次十连抽抽奖平均可获得：
- UR道具：1个
- SSR道具：11个
- SR道具：8个
- R道具：80个

> **说明**：该数据为大量数据进行的平均测算，玩家实际获得的数据可能有所变化。

## 四、保底机制

### 4.1 传说宝箱
宝箱开启100个至少会获得1个SSR。

### 4.2 神话宝箱
宝箱开启100个至少会获得1个UR。

> **说明**：当前地图下所有同类型宝箱共享此概率数值。

## 五、作者修改规则

1. 作者可自行根据地图需求增加、删除、修改地图宝箱内的道具
2. 每次修改均会在页面内进行公告
3. 平台对作者进行的修改不进行审核
4. 每次修改至少间隔七天

## 六、重复道具处理规则

抽到已有的有限时限或限量道具将会自动为玩家叠加：

### 6.1 已有永久道具

如已有永久道具抽取到对应的限时/限量/永久道具，将为玩家自动将抽取到的道具分解为地图积分：

#### 传说宝箱分解积分

| 品质 | 分解积分 |
|------|---------|
| SSR | 400积分 |
| SR | 100积分 |
| R | 25积分 |
| N | 5积分 |

#### 神话宝箱分解积分

| 品质 | 分解积分 |
|------|---------|
| UR | 600积分 |
| SSR | 400积分 |
| SR | 100积分 |
| R | 25积分 |

### 6.2 已有时限/限量道具

如已有时限/限量道具抽取到对应的限时/限量/永久道具：
- 时限道具：延长时限
- 限量道具：增加数量
- 永久道具：自动分解为地图积分（同上表）

## 七、抽取奖励积分

每次抽取宝箱可额外获得典藏积分奖励，积分可用于兑换稀有道具：

### 7.1 传说宝箱奖励积分

| 品质 | 奖励积分 |
|------|---------|
| N | 10积分 |
| R | 50积分 |
| SR | 200积分 |
| SSR | 1000积分 |

### 7.2 神话宝箱奖励积分

| 品质 | 奖励积分 |
|------|---------|
| R | 50积分 |
| SR | 200积分 |
| SSR | 1000积分 |
| UR | 3000积分 |

> **说明**：宝箱道具均为不可堆叠类型，每个账号最多拥有1个。

## 八、积分用途

分解获得的地图积分可用于：
- 在积分商城兑换道具
- 购买特殊宝箱
- 兑换限时奖励

## 九、积分商城兑换价格

玩家可使用地图积分直接兑换对应品质的道具：

| 品质 | 兑换所需积分 |
|------|------------|
| N | 288积分 |
| R | 588积分 |
| SR | 1288积分 |
| SSR | 2888积分 |
| UR | 5000积分 |

> **说明**：积分兑换的道具与抽奖获得的道具属性完全一致，玩家可根据需求选择直接兑换或抽奖获取。
