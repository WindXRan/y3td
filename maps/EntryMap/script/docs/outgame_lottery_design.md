# 游戏外抽奖宝箱系统设计文档

## 1. 需求分析

### 1.1 业务背景
根据项目需求，需要设计两个游戏外抽奖宝箱，参考KK平台地图的道具设计风格，每个道具获得时需增加具体属性。

### 1.2 核心需求
| 需求编号 | 需求描述 | 来源 |
|---------|---------|------|
| REQ-001 | 设计两个游戏外抽奖宝箱（普通宝箱、高级宝箱） | 用户需求 |
| REQ-002 | 宝箱道具需带有具体属性加成效果 | 用户需求 |
| REQ-003 | 参考KK平台地图道具设计风格 | 用户需求 |
| REQ-004 | 支持概率权重控制掉落 | 系统设计 |
| REQ-005 | 高级宝箱需有保底机制 | 系统设计 |

### 1.3 道具属性体系（参考KK平台）
| 属性名称 | 单位 | 说明 |
|---------|------|------|
| 攻击加成 | % | 提升角色攻击百分比 |
| 生命加成 | % | 提升角色生命百分比 |
| 最终伤害 | % | 提升最终伤害输出 |
| 能量 | 数值 | 技能能量值 |
| 神识 | 数值 | 神识属性值 |
| 魂力 | 数值 | 魂力属性值 |
| 金币 | 数值 | 游戏货币 |
| 木材 | 数值 | 养成材料 |
| 石材 | 数值 | 养成材料 |
| 排名积分 | 数值 | PVP排名积分 |

## 5. 概率计算逻辑

### 5.1 加权随机算法
1. 计算奖励池总权重 `total_weight`
2. 生成 0 ~ total_weight 的随机数 `roll`
3. 遍历奖励池，累加权重，当累加值 >= roll 时选择该道具

### 5.2 保底机制（高级宝箱）
- 十连抽时，至少保证获得1个稀有或更高品质道具
- 若前9次均为普通品质，第10次强制从稀有+品质池中抽取

## 8. 设计原则

1. **差异化定位**：普通宝箱面向日常福利，高级宝箱面向深度养成玩家
2. **概率透明**：权重系统可配置，便于数值调整
3. **保底保障**：高级宝箱十连保底提升抽卡体验
4. **数值梯度**：高级宝箱奖励为普通宝箱的2-3倍
5. **收集驱动**：包含英雄碎片，鼓励长期积累
6. **PVP激励**：高级道具包含排名积分奖励
