# Codex Y3 Snapshot

This directory is a sanitized project snapshot of the local Codex Y3 setup.

Included:
- `config.toml`
- `mcp_settings.y3maker.json`
- Y3-related Codex skills
- Migrated Y3 knowledge, rules, and memory files

Excluded:
- `auth.json`
- `cap_sid`
- installation and session identifiers
- SQLite log/state databases
- sandbox/session/tmp directories
- generated caches and runtime queues
