# 2026-04-07 自动主动效果扩充设计

## 目标

在不新增玩家手动按键、不重做战斗主系统的前提下，为当前局内玩法补一批“来源于羁绊、宝物、烙印的自动主动效果”，让战斗表现更充实、局内更像“角色一直在放技能”。

本轮目标偏速度，优先做最小可玩内容扩充，不追求完整新系统。

## 本轮范围

- 新增 `8` 个自动主动效果
- 来源仅限：
  - 羁绊
  - 宝物
  - 烙印
- 触发方式仅限：
  - 定时触发
  - 击杀触发
  - 普攻计数触发
  - 攻击技能释放后触发
- 复用现有能力：
  - 索敌
  - 投射物
  - 范围伤害
  - 减速
  - 连锁伤害
  - 粒子表现
  - 单位动画

## 不在本轮范围

- 不新增玩家手动释放按钮
- 不单独做完整“主动技能系统”
- 不做复杂的多层升级树
- 不做大规模 UI 改造
- 不单独做完整音乐系统

## 方案选择

本轮采用“轻量联动型”方案。

特点：
- 每个来源只补 `1` 个自动主动效果
- 运行时统一走一层轻量触发逻辑
- 内容先做出来，后续再决定是否往“主动技能池系统”升级

选择原因：
- 落地最快
- 风险最小
- 最适合当前“先把游戏充实感补起来”的目标

## 内容清单

### 羁绊主动效果

1. `雷链脉冲`
- 来源建议：`chain` / `shock` 相关羁绊
- 触发：每隔一段时间自动触发
- 效果：对最近目标降下一次雷击，并弹射至附近敌人
- 目标体验：补足中期清怪和“雷法正在出手”的表现

2. `霜环震荡`
- 来源建议：`cold_tide` / `fortress` 相关羁绊
- 触发：每隔一段时间自动触发
- 效果：以英雄为中心释放一圈冰霜冲击，造成范围伤害并减速
- 目标体验：强化英雄站场存在感

3. `灼羽齐射`
- 来源建议：`burn` / `barrage` 相关羁绊
- 触发：每隔一段时间自动触发
- 效果：朝前方扇形打出多枚火焰弹
- 目标体验：让前方持续有技能飞行物和命中反馈

### 宝物主动效果

4. `守护脉冲`
- 来源建议：防御/生存向宝物
- 触发：周期性触发
- 效果：为英雄生成短时护盾，并附带明显释放表现
- 目标体验：补足“宝物也在参与战斗”的感知

5. `收割飞刃`
- 来源建议：输出/击杀向宝物
- 触发：击杀后触发
- 效果：生成追踪飞刃，对附近敌人进行补刀或追击
- 目标体验：让连杀过程更热闹

6. `金币爆鸣`
- 来源建议：资源/贪婪向宝物
- 触发：周期性或低血线条件触发
- 效果：对低血量或密集目标触发一次小范围爆炸
- 目标体验：让经济向宝物也有战斗表现

### 烙印主动效果

7. `血怒践踏`
- 来源建议：近战/站场/怒气向烙印
- 触发：累计若干次普攻后触发
- 效果：以英雄脚下释放震击波，造成小范围伤害或短控制
- 目标体验：强化近战英雄每轮普攻循环后的爆点

8. `星火回响`
- 来源建议：技能联动向烙印
- 触发：任意攻击技能释放后触发
- 效果：追加一发自动追击弹或小爆点
- 目标体验：放技能后有明显“回响”感

## 实现结构

### 静态层

新增一组“自动主动效果”静态定义，字段尽量简单：

- `effect_id`
- `name`
- `source_type`
- `source_id`
- `trigger_type`
- `cooldown`
- `counter_required`
- `target_rule`
- `damage_ratio`
- `radius`
- `vfx_key`
- `sfx_key`

静态表按来源类型拆分或统一收口都可以，但首轮更推荐统一收口，便于运行时遍历。

### 运行时

新增轻量触发层，统一负责：

- 判断当前是否已拥有来源
- 维护每个效果的冷却/计数
- 在战斗循环里更新触发状态
- 命中条件时调用现有战斗能力执行

首轮不做复杂对象系统，优先采用“定义表 + runtime state”的形式。

### 复用能力

尽量复用现有攻击技能运行时已有的能力：

- 获取目标
- 获取范围敌人
- 发射投射物
- 结算伤害
- 结算减速/控制
- 播粒子
- 播单位动画

这样可以避免重复造一套技能执行链路。

## 表现策略

本轮表现以“够明显”为标准，不追求豪华资源。

- 每个自动主动效果至少要有：
  - 触发粒子或命中特效
  - 明显的伤害/减速/弹射反馈
- 能复用现有投射物与粒子的先复用
- 新音效只跟着主动效果补最基础的一轮，不单独开音乐工程

## 平衡边界

首轮数值原则：

- 单个自动主动效果不应压过主攻击技能
- 更偏“补表现 + 补节奏”
- 触发频率宁可偏保守，后续再提高
- 羁绊效果强于普通宝物效果
- 烙印效果偏联动，不单独承担主输出

## 风险与控制

主要风险：
- 自动效果太多导致战场噪音过高
- 运行时触发条件分散，后面不好维护
- 伤害叠加过快，压缩主技能存在感

控制方式：
- 首轮只做 `8` 个
- 统一触发层
- 统一冷却与计数字段
- 数值先保守

## 推荐落地顺序

1. 先建自动主动效果静态定义
2. 接运行时统一触发层
3. 先做 `雷链脉冲`、`霜环震荡`、`守护脉冲`
4. 再补其余 `5` 个
5. 最后补基础音效与粒子绑定

## 预期结果

完成后，玩家即使不额外按键，也会在中局开始明显感受到：

- 羁绊会“主动出手”
- 宝物会“参与战斗”
- 烙印会“打出节奏点”
- 英雄周围持续有技能释放、投射物、范围冲击和追击效果

这会明显提升当前版本的充实感和战斗热闹程度。
