# 羁绊修仙包装命名表

> 传播口径一句话：
> 把“羁绊系统”说成“仙缘感应、参悟道统、炼化机缘”的修行成长线。

> 原始系统说明、旧命名口径、结构规则，仍保留在 `08-羁绊系统策划案.md`。
> 本文更偏对外命名、UI口径、文案传播，不替代原稿。

## 一句话版本

- 玩家通过 `F 仙缘感应` 结下机缘。
- 补齐一路后，悟得该脉的 `道统真意`。
- 成型后的机缘会被 `炼化`，化作自身底蕴。

## 可直接转发的三句话

1. 羁绊不再是“凑套装”，而是“感应仙缘、参悟道统”。
2. 玩家拿到的不是“卡”，而是“缘”；触发的不是“套装效果”，而是“道统真意”。
3. 六大方向统一包装为：`炼体 / 财缘 / 玄法 / 弓修 / 诛心 / 修行`。

## 命名总规则

- 四字名只留给最高级羁绊。
- 根脉和中段分支优先压到 `2-3` 字，方便玩家喊、记、传播。
- 终局级、上位级、最有记忆点的路线，才使用四字名来抬规格。

一句话理解：

- `低阶短，终局长；中段利落，顶级显名。`

## 六大方向

这六个词是最核心、最适合传播的一级分类口径。

- 炼体：
  - 原称呼：`体术`
  - 推荐感受：肉身、护体、血气、破阵
  - 根脉名：`金刚体`
  - 代表路线：`长生 / 兵法 / 不动 / 破军 / 血誓 / 血魔`
- 财缘：
  - 原称呼：`经济`
  - 推荐感受：财运、资源、机缘、福报
  - 根脉名：`天府缘`
  - 代表路线：`吞财 / 渡劫`

  - 原称呼：`法术`
  - 推荐感受：御法、行炁、五行、雷爆
  - 根脉名：`玄门法`
  - 代表路线：`金灵 / 玄炁 / 爆法 / 迅诀 / 厚土`
- 弓修：
  - 原称呼：`箭术`
  - 推荐感受：穿云、百羽、逐日、疾风
  - 根脉名：`天弓道`
  - 代表路线：`广射 / 疾风 / 穿云`
- 诛心：
  - 原称呼：`暴击`
  - 推荐感受：一击灭形、破妄诛心、杀机毕露
  - 根脉名：`诛心诀`

- 修行：

  - 推荐感受：修身、修力、修法、循序登仙

## 命名风格原则

这套包装要尽量做到“看得懂、记得住、说得出”。

- 先让玩家一眼能懂，再追求仙气。
- 优先 `2-4` 字短词，最好能被玩家直接喊出来。
- 四字名只给最高级：
  - 例如 `显圣真君 / 天师真传 / 神弓真传 / 红莲魔君`
- 根脉和中段分支尽量保持 `2-3` 字：
  - 例如 `天府缘 / 金灵 / 厚土 / 伏魔 / 灵台`
- 一级分类要像修行方向：
  - 例如 `炼体 / 财缘 / 玄法 / 弓修 / 诛心 / 修行`
- 根脉命名要像一条完整道统：
  - 例如 `金刚体 / 天弓道 / 玄门法 / 诛心诀`
- 上位路线命名要像人物、法门或真传：
  - 例如 `伏魔 / 显圣真君 / 天师真传`
- 系统动作优先用动词：
  - `感应 / 参悟 / 悟得 / 炼化 / 收下此缘`
- 尽量少用偏西幻或偏数值策划味道的词：
  - 例如把 `猎魔人 / 大炮 / 元素师` 改成 `伏魔 / 天诛法 / 厚土`
