# 十五个新版初始技能设计总表

> 本文档对应当前工程内已入库的 `15` 个新版初始技能蓝图。
>
> 风格统一为 `修仙 / 西游` 包装，技能池目标是替代旧的少量固定初始技能方案，形成更大的局内构筑池。

## 文档定位

- 面向当前版本的新版初始技能池
- 作为策划、UI、运行时统一阅读口径
- 对应工程落点：
  - `data_csv/attack_skill_second_batch_skills.csv`
  - `data_csv/attack_skill_second_batch_evolutions.csv`
  - `data_csv/attack_skill_second_batch_cards.csv`
  - `runtime/attack_upgrades.lua`
  - `runtime/attack_skills.lua`

## 统一规则

- 每个技能固定采用 `3 张普通 + 3 张优秀 + 3 张稀有 + 1 张终局` 的 `9+1` 结构。
- `普通` 负责伤害、冷却、核心功能。
- `优秀` 负责范围、数量、形态变化。
- `稀有` 负责状态、对精英、特殊触发。
- `传奇` 负责终局进化与最终形态。

## 统一物品说明格式

- 主标题：技能名
- 副标题：技能定位
- 属性行：伤害标签、伤害系数、冷却时间、施法范围，以及穿透/范围/持续/弹射等关键字段
- 说明行：
  - `技能说明`
  - `终局进化`
  - `装配效果`

## 统一成长节点建议

- `2级`：先解锁核心功能手感
- `5级`：范围或数量开始成型
- `7级`：基础数值获得第一次明显补正
- `10级`：形态牌开始接入
- `13级`：终局牌进入可解锁区间
- `16级`：基础冷却再次缩短
- `20级`：对精英或 Boss 特化成型
- `23级`：触发牌完成技能个性
- `26级`：核心维度再次补强
- `30级`：终局牌进入完全体
