# 归档文档入口

> 本目录用于集中收纳旧版拆分文档，默认不作为项目主阅读入口。

## 文档目标

- 保留历史设计拆分、内容池明细、模块工程说明与阶段计划记录。

- 包含：旧 `design`、旧 `内容设计`、旧 `项目模块`、旧 `开发进度与计划`。
- 不包含：当前推荐阅读顺序与主文档维护入口。
