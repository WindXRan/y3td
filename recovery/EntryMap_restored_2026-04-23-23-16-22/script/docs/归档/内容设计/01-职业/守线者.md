# 守线者

> `守线者` 是首发职业池中的 `入门模板` 职业，负责提供最稳、最直观的站场生存体验。

## 1. 职业定位

- 模板档位：`入门模板`
- 玩家标签：`易上手`
- 核心爽点：
  - 扛得住
  - 顶得稳
  - 越站越强
- 适合人群：
  - 新玩家
  - 喜欢稳扎稳打的玩家
  - 希望低投入稳定推进内容的玩家

## 2. 核心机制

- 以 `站场生存`、`受击收益`、`反击`、`吸血` 为主要玩法支柱。
- 守线者的核心体验不是瞬间爆发，而是：
  - 在高压环境下不容易崩盘
  - 靠稳定近战输出和受击反馈慢慢滚起优势
- 这个职业要教会玩家：
  - 活下来本身就是收益
  - 站住位置就能把一局慢慢打顺

## 3. 成长曲线

### 3.1 前期
- 依赖基础坦度和近战稳定输出站稳第一轮怪潮。
- 不需要复杂 build 就能获得明显正反馈。

### 3.2 中期
- 开始通过吸血、减伤、反击和范围清怪形成稳态。
- 当站场能力建立后，整体推进会明显变顺。

### 3.3 后期
- 依赖持续站场和抗压能力完成终盘检定。
- 强势点在于容错和持续作战，不在极限爆发。

## 4. 流派分支

### 4.1 反击壁垒流
- 关键词：
  - `受击触发`
  - `反伤`
  - `格挡`
  - `减伤`
- 玩法重点：
  - 让敌人打你本身也变成收益
  - 适合高压怪潮、频繁受击场景

### 4.2 血怒站场流
- 关键词：
  - `吸血`
  - `生命上限`
  - `近战范围`
  - `持续作战`
- 玩法重点：
  - 通过更高的回复和站桩收益维持长时间作战
  - 更强调“我不倒，防线就不倒”

## 5. 天赋树方向

### 5.1 基础层
- 生命
- 防御
- 恢复
- 站场稳定性

### 5.2 分支层
- `反击强化`
  - 受击收益提升
  - 反击频率和强度提升
- `吸血强化`
  - 回复效率提升
  - 持续作战能力提升

### 5.3 核心层
- 低血不崩盘
- 站桩越久越强
- 近战对群能力质变

## 6. 推荐标签

### 6.1 推荐装备标签
- `生命`
- `护甲`
- `减伤`
- `吸血`
- `近战范围`

### 6.2 推荐遗物标签
- `生存`
- `持续作战`
- `受击收益`

## 7. 设计边界

- 不能同时成为最肉和最高输出职业。
- 主要优势应始终是：
  - 高容错
  - 高稳定性
  - 不容易崩盘
- 如果后期输出过高，会挤压追猎者、蚀骨术士等职业的空间。
