# 蚀骨术士

> `蚀骨术士` 是首发职业池中的 `主力模板` 职业，承担稳定发育、持续成长、适合平民主养的中坚角色。

## 1. 职业定位

- 模板档位：`主力模板`
- 玩家标签：`推荐主力`
- 核心爽点：
  - 挂上状态后越打越疼
  - 层数滚起来后压力持续扩大
  - 成型后对群效率很高
- 适合人群：
  - 喜欢稳定 build 和长期成长的玩家
  - 希望平民主养但又不想玩太直白职业的玩家

## 2. 核心机制

- 以 `持续伤害`、`诅咒`、`叠层`、`扩散` 为核心。
- 蚀骨术士的强势点不在瞬间爆发，而在：
  - 持续给敌人施压
  - 通过层数和联动慢慢接管战场
- 玩这个职业的核心心态是：
  - 先铺状态
  - 再吃联动
  - 最后让整波怪自己崩掉

## 3. 成长曲线

### 3.1 前期
- 伤害起步不算最炸，但挂状态的节奏稳定。
- 初期体验偏“铺垫”，不是立刻大爆发。

### 3.2 中期
- 叠层和扩散开始起作用后，清怪效率明显上升。
- 这时 build 会开始显出强度差异。

### 3.3 后期
- 对群持续压血能力极强。
- 一旦状态体系完整，整局输出会非常稳定。

## 4. 流派分支

### 4.1 腐蚀叠层流
- 关键词：
  - `叠层`
  - `持续伤害`
  - `高层收益`
- 玩法重点：
  - 让单个目标或一批目标进入越来越痛的状态
  - 更强调厚血怪和 Boss 处理能力

### 4.2 诅咒扩散流
- 关键词：
  - `扩散`
  - `连锁传播`
  - `对群压制`
- 玩法重点：
  - 击杀、触发或扩散后让整波怪一起吃状态
  - 更强调清杂和波次推进

## 5. 天赋树方向

### 5.1 基础层
- 持续伤害
- 效果持续时间
- 状态命中
- 技能稳定性

### 5.2 分支层
- `叠层做深`
  - 高层诅咒收益
  - 层数成长更陡
- `扩散做深`
  - 传播效率
  - 对群蔓延速度

### 5.3 核心层
- 高层诅咒质变
- 持续伤害联动收益
- 对群传播后的收割能力

## 6. 推荐标签

### 6.1 推荐装备标签
- `持续伤害`
- `技能伤害`
- `效果持续`
- `范围`

### 6.2 推荐遗物标签
- `DOT强化`
- `诅咒收益`
- `扩散收益`

## 7. 设计边界

- 不能在前期就比追猎者更直接爽。
- 它的优势必须体现在中后期成型后的稳定压制。
- 如果持续伤害和爆发同时过强，会让职业失去节奏差异。
