# 御灵使

> `御灵使` 是首发职业池中的“强力主力”，属于 `主力模板`，主打召唤、控场与场面压制。

## 1. 职业定位

- 模板档位：`主力模板`
- 玩家标签：`推荐主力`
- 核心爽点：
  - 场上单位越来越多
  - 防线越来越稳
  - 场面感最强
- 适合人群：
  - 喜欢召唤体系和控场体系的玩家
  - 喜欢“自己带着一套体系打”的玩家

## 2. 核心机制

- 以 `召唤物`、`协同输出`、`控场`、`前后排分工` 为主要玩法结构。
- 御灵使不是自己一个人把伤害打满，而是：
  - 召唤体系一起承担输出
  - 召唤体系一起维持场面控制
- 这个职业要给玩家的核心体验是：
  - 我不是一个人守线
  - 我的体系越完整，战场越听我指挥

## 3. 成长曲线

### 3.1 前期
- 召唤物能够尽早分担一部分压力。
- 上手阶段就能感受到“整体战斗变轻松”的收益。

### 3.2 中期
- 召唤物数量、质量和联动逐渐成型。
- 场面会明显变大，守线压力变得更可控。

### 3.3 后期
- 依靠协同输出和控场形成持续压制。
- 强项在整体战局掌控，而不是单点击杀。

## 4. 流派分支

### 4.1 兽群压制流
- 关键词：
  - `数量`
  - `协同`
  - `覆盖`
- 玩法重点：
  - 通过更多单位覆盖战场
  - 强调成片压制和推进

### 4.2 精英御灵流
- 关键词：
  - `高质量召唤物`
  - `协战`
  - `单体强压`
- 玩法重点：
  - 少量强召唤体承担主力输出
  - 更适合 Boss 和关键目标处理

### 4.3 控场指挥流
- 关键词：
  - `减速`
  - `牵制`
  - `区域控制`
- 玩法重点：
  - 让召唤物和本体共同维持战线秩序
  - 更强调防线安全和节奏掌控

## 5. 天赋树方向

### 5.1 基础层
- 召唤物生命
- 召唤物攻击
- 持续时间
- 数量稳定性

### 5.2 分支层
- `数量体系`
  - 更多单位、更高覆盖
- `质量体系`
  - 更强个体、更高协战强度
- `控场体系`
  - 控制效果、更稳战线

### 5.3 核心层
- 召唤协战质变
- 召唤连锁触发
- 控场体系完成后的全局稳定性提升

## 6. 推荐标签

### 6.1 推荐装备标签
- `召唤强化`
- `持续时间`
- `范围`
- `控场`

### 6.2 推荐遗物标签
- `召唤收益`
- `协同输出`
- `场面压制`

## 7. 设计边界

- 不能让召唤物完全盖过玩家本体存在感。
- 不能强到自动解决所有关卡问题。
- 其优势应体现在：
  - 整体战局更顺手
  - 场面更稳定
  - build 成型后更容易打出“大场面爽感”
