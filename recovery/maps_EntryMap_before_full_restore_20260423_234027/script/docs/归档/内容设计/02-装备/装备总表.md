# 装备总表

> 本文用于管理局外装备内容池；装备系统原则以 `docs/design/初步设计/03-局外成长系统.md` 为准。

## 1. 当前定案摘要

- `装备` 有品质，使用 `普通（白）/ 优良（绿）/ 精良（蓝）/ 史诗（紫）/ 传奇（橙）/ 传说（金）` 六档体系。
- `装备` 主要通过 `刷关掉落` 获取，是长期毕业主来源。
- `装备` 采用 `随机主词条 + 随机副词条 + 随机数值档位` 结构。
- `装备` 支持 `洗练 / 重铸`。
- `装备` 可以提供：
  - 基础词条
  - 副词条
  - 套装效果
- 套装效果允许包含 `小幅机制强化`，但总体权重低于高稀有遗物。
- 卡池可投放少量 `通用型强力装备`，但同品质下应弱于刷图定向装备约 `20%`。

## 2. 建议记录字段

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| `equipment_set_id` | 套装 ID | 装备所属套装 |
| `display_name` | 名称 | 套装或装备显示名 |
| `slot_type` | 部位 | 首版可按 `5 部位` 规划 |
| `quality_pool` | 可出现品质 | 控制不同阶段产出 |
| `main_stat_pool` | 主词条池 | 按部位与职业倾向定义 |
| `sub_stat_pool` | 副词条池 | 随机候选类型 |
| `roll_tier_rule` | 数值档位规则 | 控制高低档区间 |
| `set_effect_2` | 两件套效果 | 可为数值或轻机制 |
| `set_effect_4` | 四件套效果 | 可为更明确方向的强化 |
| `drop_stage_tier` | 掉落层级 | 对应章节或难度 |
| `generic_pool_flag` | 是否通用卡池装备 | 用于区分刷图定向装与卡池装 |
| `status` | 当前状态 | `设计中 / 未实现 / 部分实现 / 已实现` |
