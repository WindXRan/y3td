# 装备部位规则

> 本文定义局外装备的部位结构与每个部位的大致职责，不填写具体装备内容。

## 1. 当前定案

- 局外装备采用 `5 部位` 结构，参考“圣遗物式”换装体验。
- 每个部位都允许拥有：
  - `品质`
  - `主词条`
  - `副词条`
  - `套装归属`
- 不同部位的主词条池应存在差异，用于控制 build 方向与掉落价值。

## 2. 建议部位

| 部位 | 设计职责 | 说明 |
| --- | --- | --- |
| `武器位` | 核心输出倾向 | 更容易承载攻击、技能、暴击、攻速等强进攻词条 |
| `头部位` | 平衡型补强 | 适合承载命中、生存、功能型主词条 |
| `胸甲位` | 防御与站场 | 适合承载生命、防御、减伤等稳定属性 |
| `饰品位` | 功能与触发 | 适合承载冷却、资源回复、特殊收益类词条 |
| `徽记位` | 流派放大器 | 适合承载职业倾向、流派标签和套装关键拼图 |

## 3. 部位设计原则

- `武器位` 不应一枝独秀到压过所有其他部位。
- `胸甲位` 应更偏防线稳定，而不是纯堆伤害。
- `饰品位` 和 `徽记位` 应更多承担“build 定向”和“特殊收益”职责。
- 每个部位都应能成为玩家判断“这件装备有没有价值”的独立维度。
