# 三十名国风英雄图鉴与专属神通

> 本文档是新开的英雄包装文档。
>
> 它不替代已有旧文档，而是在当前工程内新增一套 `30` 名英雄的 `修仙 / 西游 / 国风` 图鉴口径，并把每名英雄的 `专属神通` 正式落到代码配表里。

## 对应工程落点

- `data_csv/hero_roster.csv`
- `data_csv/hero_form_skills.csv`
- `data/object_tables/hero_roster.lua`
- `data/object_tables/hero_form_skills.lua`
- `runtime/hero_form_skills.lua`

## 使用规则

- 当前实现已切到库内模型，并生成独立英雄承载单位。
- 英雄专属神通通过 `当前英雄形态 unit_id` 自动识别。
- 已存在的进化换模会直接吃到这套神通映射。
- 稀有度使用 `R / SR / SSR / UR` 四档。

## 神通说明格式

- 主标题：神通名
- 副标题：战斗定位
- 触发行：几次普攻、几次攻击技能、几次击杀或固定秒数
- 效果行：伤害类型、范围、弹射、追击、回复、处决、金币等核心字段

一句话理解：

- `英雄负责身份包装`
- `神通负责战斗个性`
- `当前英雄形态负责把两者接起来`

## 按稀有度查看

### R

- `赤髯罗汉 / 开山吼`：普攻叠满后震开近身怪。
- `玄甲都统 / 铁城印`：定时护体并回复生命。
- `瑶池仙姝 / 水月绫`：攻击技能循环后追加水绫回斩。
- `巡海枪使 / 断潮刺`：单点突刺后炸出小范围水波。
- `玉笛药仙 / 回春曲`：击杀节奏达到后回复并反击周边。
- `流云剑侯 / 追云斩`：普攻后续追斩多个目标。
- `铁壁山君 / 不动印`：稳定型守线脉冲。
- `黄天法王 / 黄天雷`：符雷弹射适合清杂兵。
- `踏浪夜叉 / 怒潮刃`：尸爆加金币，适合滚雪球。
- `桃华仙姝 / 花月潮`：较温和的回复与清线一体化。
- `破阵刀卫 / 横江断`：贴脸清线用的近战横斩。
- `独目战魄 / 裂虹斩`：专门补掉残血怪与残血精英。

### SR

- `天机灵女 / 千机翎`：机关多段点杀。
- `霹雳弩姬 / 霹雳弩`：普攻触发雷链，传播感强。
- `九黎战将 / 裂地踏`：靠大范围践踏站住前线。
- `云渊少君 / 白龙破`：长链穿杀，穿阵速度快。
- `赤羽天弓 / 连珠箭`：技能后追加扇面齐射。
- `月华仙姝 / 霜华舞`：短间隔多轮连打。
- `白虹枪仙 / 白虹贯`：技能后补出高额单点穿刺。
- `镇岳金刚 / 金刚震`：厚重型普攻爆发。
- `青莲君子 / 青莲护`：低血自动回春的保命位。

### SSR

- `沧澜战魂 / 沧海怒`：连续海潮斩击，压场感强。
- `逐日神弓 / 落日矢`：非常明确的远程斩首定位。
- `火德真君 / 赤霄焰`：技能循环中不断落焰轰场。
- `灵台天师 / 七星灯`：长时间多轮点杀与压制。
- `太虚谋圣 / 夺魄局`：技能后对中后排多目标追杀。
- `玄锋真君 / 青龙偃`：大范围重型刀芒清场。

### UR

- `显圣真君 / 天目戟`：高频多轮连斩，最纯粹的压制型 UR。
- `幽都天相 / 噬魂符`：尸体会继续变成伤害来源，滚雪球最强。
- `北辰帝君 / 天命令`：帝令降场，适合处理中轴密集敌群。

## 当前版本落地方式

- 这套图鉴目前优先服务 `英雄形态切换`。
- 只要英雄模型被切到上述 `unit_id`，战斗里就会自动启用对应神通。
- 也就是说：
  - 当前已有的进化换模功能，可以直接映射到专属神通。
  - 后续如果你继续扩更多进化、局外招募或大厅选将，也可以直接复用这两张表。
