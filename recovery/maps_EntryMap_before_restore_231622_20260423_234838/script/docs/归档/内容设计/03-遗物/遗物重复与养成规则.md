# 遗物重复与养成规则

> 本文定义遗物重复获取后的处理方式和长期养成框架。

## 1. 重复获取的目标

- 避免抽到重复遗物后“完全没用”。
- 把重复遗物转化为：
  - 长期成长资源
  - 保底进度
  - 兑换价值

## 2. 重复处理建议

- 重复遗物可转化为：
  - `遗物碎片`
  - `升阶材料`
  - `商店兑换货币`
- 转化价值应与品质挂钩，高品质重复应更有价值。

## 3. 遗物养成建议

- 遗物可预留以下长期养成方向：
  - `升星`
  - `升阶`
  - `共鸣等级`
- 养成目标应偏“数值放大”和“已有机制强化”，而不是新建独立机制。

## 4. 设计边界

- 重复转化的收益不能低到让抽卡体验崩坏。
- 养成收益不能高到让遗物系统压倒装备系统。
- 同一遗物的养成应有上限，避免无限堆叠。
