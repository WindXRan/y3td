# 掉落与卡池总表

> 本文用于管理装备掉落、遗物卡池、保底规则、商店池和其他长期获取入口。

## 1. 文档职责

- 记录不同章节和难度的装备掉落层级。
- 记录遗物卡池的品质分布、池子类型与保底规则。
- 记录通用型卡池装备、商店池和活动池的投放边界。

## 2. 建议记录字段

| 字段 | 含义 | 说明 |
| --- | --- | --- |
| `pool_id` | 池子 ID | 掉落池、卡池、商店池统一标识 |
| `pool_type` | 池子类型 | `关卡掉落 / 常驻卡池 / 限定卡池 / 商店池` |
| `target_content` | 目标内容 | 装备、遗物、材料等 |
| `quality_distribution` | 品质分布 | 控制各品质出现概率 |
| `stage_or_mode_gate` | 解锁条件 | 章节、难度、模式要求 |
| `currency_type` | 消耗货币 | 金币、钻石、活动代币等 |
| `pity_rule` | 保底规则 | 主要用于遗物卡池 |
| `duplicate_rule` | 重复处理规则 | 分解、转化、兑换 |
| `status` | 当前状态 | `设计中 / 未实现 / 部分实现 / 已实现` |

## 3. 当前边界

- `金币不能进入高级卡池`。
- `钻石` 是主要卡池货币，但免费投放要明显存在。
- `装备毕业` 仍以刷图体系为主，卡池中的通用装备只承担补进度和泛用战力职责。
- `橙 / 金遗物` 是卡池长期追求点，但不能成为基础游玩门槛。
