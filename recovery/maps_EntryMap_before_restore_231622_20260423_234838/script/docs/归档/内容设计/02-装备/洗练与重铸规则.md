# 洗练与重铸规则

> 本文定义装备洗练与重铸的框架，不写具体消耗数值。

## 1. 目标

- 让随机装备存在长期追求。
- 让玩家在“继续刷”与“投入资源修正”之间做选择。
- 避免无限低成本洗到毕业。

## 2. 基础洗练

- 基础洗练用于重新随机装备的部分词条结果。
- 建议职责：
  - 处理“这件装备方向对，但词条不够好”的情况
  - 主要重置副词条结果
- 主要消耗：
  - `金币`
  - `普通洗练材料`

## 3. 高级重铸

- 高级重铸用于更高价值的定向修正。
- 建议职责：
  - 锁定部分词条
  - 重置其他词条
  - 处理高品质装备的毕业追求
- 主要消耗：
  - `钻石侧稀有材料`
  - `高级重铸材料`

## 4. 设计原则

- 低品质装备的修正成本不应过高。
- 高品质装备的毕业修正应足够贵，避免轻易毕业。
- 洗练系统的目标是“修方向”，不是免费制造完美装备。
